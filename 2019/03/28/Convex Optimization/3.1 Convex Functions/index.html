3. Convex Functions

3.1 Basic properties and examples

3.1.1 Definition

A function \(f: \RR^n\rightarrow \RR\) is convex if \(\dom f\) is a convex set and if for all \(x,y\in\dom f\), and for \(0\le\theta\le 1\) we have $$

f(x+(1-)y)f(x)+(1-)f(y) $$ A function is strictly convex if strict inequality holds when ever \(x\neq y\) and \(0<\theta<1\). We say \(f\) is concave if \(-f\) is convex. For an ==affine function==, we always have the equality of the above definition, so all affine functions are both ==convex and concave==. Conversely, any function that is convex and concave is affine.

A function is ==convex== if and only if it is convex when restricted to any line that intersects its domain. In other words \(f\) is convex if and only if for all \(x\in \mathbf{dom}\space f\) and all \(v\), the function \(g(t)=f(x+tv)\) is convex.

3.1.2 Extended-value extensions

It is often convenient to extend a convex function to all of \(\mathbf{R}^n\) by defining its value to be infinity outside its domain. \[ \tilde f(x)= \begin{cases} f(x)\quad x\in dom\space f\\ \infty\quad \notin dom\space f \end{cases} \] The extension is defined on all \(\mathbf{R}^n\), the extension can simplify notation since we do not need to explicitly describe the domain.

==Example== Indicator function of a convex set

Let \(C\subseteq \mathbf{R}^n\) be a convex set, and consider the convex function \(I_C\) with domain \(C\) and \(I_C(x)=0\space \forall x\in C\). In other words, the function is identically zero on the set \(C\). Its extended value extensions is given by \[ \tilde I_C(x)= \begin{cases} 0 & x\in C\\ \infty & x\notin C \end{cases} \] We can play several notation tricks with the indicator function \(I_C\). For example, the problem of minimizing a function \(f\) on the set \(C\) is the same as minimizing the function \(f+\tilde I_C\) over all of \(R^n\).

3.1.3 First-order conditions

Suppose \(f\) is differentiable, Then \(f\) is convex if and only if \(\mathbf{dom}\space f\) is convex and \[ f(y)\ge f(x)+\nabla f^T(x)(y-x) \] holds for all \(x,y \in \mathbf{dom}\space f\).

==Strict convexity== can also be characterized by a first-order condition. \[ f(y)> f(x)+\nabla f^T(x)(y-x) \] ==Proof== of the first-order convexity condition

First we prove the \(n=1\) case.

We show that a differentiable function is convex if and only if \[ f(y)\ge f(x)+f'(x)(y-x) \] for all \(x,y \in \mathbf{dom}\space f\).

To prove the necessity, assume \(f\) is convex and \(x,y \in \mathbf{dom}\space f\). Since \(\mathbf{dom}\space f\) is convex, we have for all \(0<t\le 1\), \(x+t(y-x) \in dom \space f\), since \(f\) is convex, we have \[ f(x+t(y-x))\le tf(y)+(1-t)f(x) \] Then we have \[ f(y)\ge\frac{f(x+t(y-x))-f(x)}{t}+f(x) \] Taking the limit and we have the theorem.

To show the sufficiency, assume all \(x,y \in \mathbf{dom}\space f\). Choose any \(x\neq y\) and \(0\le \theta\le 1\) and let \(z=\theta x+(1-\theta)y\) \[ f(x)\ge f(z)+f'(z)(z-x)\quad f(y)\ge f(z)+f'(z)(z-y) \] Multiplying the first inequality with \(\theta\) and the second with \(1-\theta\) and adding them, then we get the proof.

The same for general case.

3.1.4 Second-order conditions

We now assume that \(f\) is twice differentiable, that is, its Hessian or second order derivative \(\nabla^2 f\) exists at each point in \(\mathbf{dom}\space f\). Then \(f\) is convex if and only if \(\mathbf{dom}\space f\) is convex and its Hessian is positive semi definite: \[ \nabla^2 f(x)\succeq 0\quad \forall x\in dom \space f \] if \(\nabla^2 f(x)\succ 0\quad \forall x\in dom \space f\), then \(f\) is strictly convex. The strictly convex converse is not true. i.e. \[ f(x)=x^4 \] is strict convex but it has zero second derivative at \(x=0\).

==Example== Quadratic functions

Consider the quadratic function \(f:\mathbf{R}^n\rightarrow \mathbf{R}\), with \(\mathbf{dom}\space f=R^n\), given by \[ f(x)=\frac{1}{2}x^TPx+q^T x+r\\ \nabla^2 f(x)=P\quad \forall x \] That means \(f\) is convex if and only if \(P\succeq0\).

==Remark== The theorem requirement that \(\mathbf{dom}\space f\) be convex cannot be dropped. For example, \[ f(x)=\frac{1}{x^2} \] is not convex since the \(\mathbf{dom}\space f\) is not convex, but \(f''(x)>0\)

3.1.5 Examples

  1. Exponential. \(e^{ax}\) is convex on \(\mathbf{R}\) for any \(a\in R\)

  2. Powers. \(x^a\) is convex on \(R_{++}\) when \(a\ge 1\) or \(a\le 0\)

  3. Powers of absolute value. \(|x|^p\) is convex on R for \(p\ge1\)

  4. Logarithm. log x is ==concave== on \(R_{++}\)

  5. Negative entropy. $x x $ is convex on \(R_{++}\)

\[ f''(x)=\frac{1}{x}>0 \]

  1. Norm. Every norm on \(R^n\) is convex. \(f:R^n\rightarrow R\) is a norm and \(0\le\theta\le1\), then

\[ f(\theta x+(1-\theta)y)\le f(\theta x)+f((1-\theta)y)=\theta f(x)+(1-\theta)f(y) \]

  1. Max function is convex on \(R^n\) . \(f(x)=\max_i x_i\) and \(0\le \theta\le 1\)

\[ f(\theta x+(1-\theta)y)=\max_i(\theta x_i+(1-\theta)y_i)\le\max_i(\theta x_i)+\max((1-\theta)y_i)=\theta f(x)+(1-\theta)f(y) \]

  1. Quadratic-over-linear function. The function \(f(x,y)=\frac{x^2}{y}\) is convex on \(R\times R_{++}\)

    To show the function is convex, we have

\[ \nabla^2f(x,y)= \begin{bmatrix} \frac{2}{y} & -\frac{2x}{y^2} \\ -\frac{2x}{y^2} & \frac{2x^2}{y^3} \end{bmatrix}=\frac{2}{y^3} \begin{bmatrix} y^2 & -xy\\ -xy & x^2 \end{bmatrix}\succeq0 \]

  1. Log-sum-exp. The function \(f(x)=\log(e^{x_1}+\dotsm+e^{x_n})\) is convex on \(R^n\).

    Define \(z=(e^{x_1},e^{x_2},\dotsm,e^{x_n})\), then \(f(z)=\log (1^T z)\). \[ \frac{\partial f(z)}{\partial z_i}=\frac{1}{1^Tz}z_i\\ \frac{\partial^2 f}{\partial z_i\partial z_j}= \begin{cases} -(1^Tz)^{-2}z_jz_i+(1^Tz)^{-1}z_{i=j} & i=j\\ -(1^Tz)^{-2}z_jz_i & i\neq j \end{cases}\\ \] That means \[ \nabla^2 f(z)=-(1^Tz)^{-2}zz^T+(1^Tz)^{-1}diag(z) \] To check if it is positive definite \[ v^T\nabla^2 f(z) v=-(1^Tz)^{-2}v^Tzz^Tv+(1^Tz)^{-1}v^Tdiag(z)v\\ =(1^Tz)^{-2}[(1^Tz)v^Tdiag(z)v-v^Tzz^Tv]\\ =(1^Tz)^{-2}\left[\left(\sum z_i \right) \left(\sum v_i^2z_i\right)-\left(\sum v_iz_i\right)^2\right]\\ \] By Cauchy-Schwarz inequality \((a^Ta)(b^Tb)\ge(a^Tb)^2\) .

    where \(a=\sqrt z_i\) and \(b=v_i\sqrt z_i\)

    We can know the above matrix is positive definite.

  2. Geometric mean. \(f(x)=(\prod_{i=1}^n x_i)^{1/n}\) is ==concave== on \(\mathbf{dom}\space f=R^n_{++}\)

    $$ =({k=1}^n x_k)^{1/n-1} = ({k=1}^n x_k)^{1/n}\

    =(1-n) (_{k=1}^n x_k){1/n}\ =({k=1}^n x_k){1/n}\ \[ Then we have \] 2f(x)=({k=1}n x_k){1/n}(qqT-ndiag(1/x_12,,1/x_n2) ) $$ where \(q_i=\frac{1}{x_i}\)

    Then for any v, we have \[ v^T \nabla^2 f(x)v =\frac{1}{n^2}(\prod_{k=1}^n x_k)^{1/n}\left(\left(\sum v_iq_i\right)^2-n\sum {v_i^2q_i^2} \right)\le 0 \]

  3. Log-determinant. For the function \(f(X)=\log \det X\) is concave.

    We can verify concavity by considering an arbitrary line, given by \(X=Z+tV\) where \(Z,V\in S^n\) \[ g(t)=f(Z+tV)=\log \det (Z+tV)\\ =\log\det(Z^{1/2}(I+tZ^{-1/2}VZ^{1/2})Z^{1/2})\\ =\log\det(I+tZ^{-1/2}VZ^{1/2})+\log \det Z\\ =\sum_i\log(1+t\lambda_i)+\log \det Z\\ g'(t)=\sum_i\frac{\lambda_i}{1+t\lambda_i}\\ g''(t)=-\sum_i\frac{\lambda_i^2}{(1+t\lambda_i)^2}<0 \] Therefore, concave.

3.1.6 Sublevel sets

The \(\alpha-sublevel\) set of a function \(f:\mathbf{R}^n\rightarrow \mathbf{R}\) is defined as \[ C_\alpha=\{x\in \mathbf{dom}\space \space f|f(x)\le\alpha\} \] The sublevel set of a convex function is convex.

3.1.7 Epigraph

The graph of a function \(f:\mathbf{R}^n\rightarrow \mathbf{R}\) is defined as \(\{(x,f(x)|x\in \mathbf{dom}\space f\}\)

The epigraph (epi means above) of a function \(f:\mathbf{R}^n\rightarrow \mathbf{R}\) is defined as \[ epi f=\{(x,t)|x\in \mathbf{dom}\space f,f(x)\le t\} \] The link between convex sets and convex functions is via the epigraph: A function is convex if and only if its epigraph is a convex set.

A function of concave if and only if its hypograph defined as \[ hypo f=\{(x,t)|t\le f(x)\} \] is a convex set.

Example

Matrix fractional function The function \(f:R^n\times S^n\rightarrow R\) ,defined as \[ f(x,Y)=x^TY^{-1}x \] is convex on \(dom f =R^n\times S^n_{++}\) . This function is the generalized quadratic over linear function. \[ epi f=\{(x,Y,t)|\space Y\succ 0, x^TY^{-1}x\le t\}\\ =\{(x,Y,t)|\space Y\succ 0, t-x^TY^{-1}x\ge 0\}\\ \] By Schur complement \[ S=t-x^TY^{-1}x\quad X= \begin{bmatrix} Y & x\\ x^T & t \end{bmatrix} \] and we have \[ \det X=\det Y\det S \] we have \[ epi f=\{(x,Y,t)|\space Y\succ 0, \begin{bmatrix} Y & x\\x^T & t \end{bmatrix}\succeq 0\}\\ \] The conditions are linear matrix inequality in (x,Y,t) and therefore, $epi f $ is convex.

3.1.8 Jensen's inequality and extensions

The basic inequality \[ f(\theta x+(1-\theta)y)\le\theta f(x)+(1-\theta)f(y) \] is sometimes called ==Jensen's inequality== . It is easily extended to convex combinations of more than two points. \[ f(\theta_1x_1+\dotsm+\theta_nx_n)\le\theta_1f(x_1)+\dotsm+\theta_nf(x_n) \] if \(f\) is convex and \(\theta_1+\dotsm+\theta_n =1\) and \(\theta_1,\theta_2,\dotsm,\theta_n\ge0\)

This is also correct in continuous cases \[ f(\int_sp(x)xdx)\le\int_sf(x)p(x)dx \] if \(f\) is convex, \(p(x)\ge0\),\(\int_sp(x)dx=1\)

In the most general case, we can take any probability measure with support in \(\mathbf{dom}\space f\) and if f is convex, we have \[ f(Ex)\le Ef(x) \]

3.1.9 Inequalities

Many famous inequalities can be derived by applying Jensen's inequality to some appropriate convex function.
