Convex Optimization Problem

4.1 Optimization Problems

4.1.1 Basic terminology

We use the notation $$

\coo [f_0(x)] {f_i(x)0,i=1,,m \h_i(x)=0,i=1,,p} $$ to describe the problem of finding an \(x\) that minimize \(f_0(x)\) among all \(x\) that satisfy the conditions \(f_i(x)\le 0,i=1,\dotsm,m\) and \(h_i(x)=0,i=1,\dotsm,p\). We call \(x\in R^n\) the ==optimization variable== and the function \(f_0: R^n\rightarrow R\) the ==objective function== or ==cost function==. The inequalities \(f_i(x)\le 0\) is called ==inequality constraints== and the corresponding functions are called ==inequality constraints functions==. The equations \(h_i(x)=0\) are called ==equality constraints== and the functions are called ==equality constraints functions==. If there is no constraint, we say the problem is ==unconstraint==.

The set of points for which the objective and all constraint functions are defined, \[ \mathcal D=\bigcap_{i=0}^m dom f_i\cap\bigcap_{i=1}^p dom h_i \] is called the ==domain== of the optimization problem. A point \(x\in \mathcal D\) is ==feasible== if it satisfies the constraints \(f_i(x)\le 0\) and \(h_i(x)=0\) . The problem is said to be ==feasible== if there exists at least one feasible point. The set of all feasible points is called a ==feasible set== of the ==constraint set==. The optimal value \(p^\star\) of the problem is defined as \[ p^\star = \inf \{f_0(x)|f_i(x)\le 0,i=1,\dotsm,m,h_i(x)=0,i=1,\dotsm,p\} \] If the problem is infeasible, \(p^\star = \infty\). If there are feasible points \(x_k\) with \(f_0(x_k)\rightarrow \infty\) as \(k\rightarrow \infty\), then \(p^\star = - \infty\) , and we say the problem is ==unbounded below==.

Optimal and locally optimal points

We say \(x^\star\) is an ==optimal point== if \(x^\star\) is feasible and \(f_0(x^\star)=p^\star\) . The set of all optimal points is the optimal set, denoted \[ X_{opt}=\{x| f_i(x)\le 0,i=1,\dotsm,m,h_i(x)=0,i=1,\dotsm,p,f_0(x)=p^\star\} \] If there exits an optimal point for the problem, we say the optimal value is ==attained== or ==achieved==, and the problem is ==solvable==. If \(X_{opt}\) is empty, we say the optimal point is ==not attained== or ==not achieved==. This always occurs when the problem is unbounded below. A feasible point x with \(f_0(x) \le p^\star +\epsilon\) is called \(\epsilon-suboptimal\) and the set of all \(\epsilon-suboptimal\) points is called the \(\epsilon-suboptimal\) set for the problem.

We say feasible point \(x\) is locally optimal (local minimizer) if there is an \(R>0\) such that \[ f_0(x)=\inf\{f_0(z)|f_i(z)\le 0,i=1,\dotsm,m,h_i(z)=0,i=1,\dotsm,p,||z-x||_2\le R\} \] roughly speaking, this means \(x\) minimizes \(f_0\) over nearby points in the feasible set. Throughout the book, optimal will mean globally optimal.

If \(x\) is feasible and \(f_i(x)=0\), we say that \(i\)th inequality constraint \(f_i(x)\le 0\) is active. otherwise it is inactive. We say that a constraint is redundant if deleting it does not change the feasible set.

Feasibility problem \[\] The feasibility problem is to determine whether the constraints are consistent, and if so, find a point that satisfies them.

4.1.3 Equivalent problems

We call two problems ==equivalent== if from a solution of one, a solution of the other is readily found. (informal definition)

We now describe some general transformations that yield equaivalent problems.

==Change of variables==

Suppose \(\phi:R^n\rightarrow R^n\) is ==one-to-one==, with image covering the problem domain \(\mathcal D\) , that means \(\mathcal D\subseteq \phi(dom \phi)\) . We define functions \(\tilde f_i\) and \(\tilde h_i\) as \[ \tilde f_i(z)=f_i(\phi(z)),i=0,\dotsm,m\\ \tilde h_i(z)=h_i(\phi(z)),i=1,\dotsm,p \] Now consider the problem $$ \coo [f_0(z)] { f_i(z)0, i=1,, m\ h_i(z)=0,i=1,,p

} $$ Then the problem is related to the original problem by \(x=\phi(z)\) . If \(z\) solves the problem, then \(x=\phi(z)\) solves the original problem.

==Transformation of objective and constraint functions==

Suppose \(\varphi_0:R^n\rightarrow R^n\) is ==monotone increasing==, \(\varphi_1,\dots,\varphi_m: R\rightarrow R\) satisfy \(\varphi_i(u)\le 0\) if and only if \(u\le 0\), and \(\varphi_{m+1},\dotsm,\varphi_{m+p}:R\rightarrow R\) satisfy \(\varphi_i(0)=0\) if and only if \(u=0\) . We define functions \(\tilde f_i\) and \(\tilde h_i\) as the compositions \[ \tilde f_i(x)=\varphi_i(f_i(x)),i=0,\dotsm,m\quad \tilde h_i(x)=\varphi_{m+i}(h_i(x)),i=1,\dotsm,p \] The associated problem \[\] and the standard form problem are equivalent; indeed the feasible sets are identical, and the optimal points are identical.

==Slack variables==

One simple transformation is based on the observation that \(f_i(x)\le 0\) if and only if there is an \(s_i\ge 0\) that satisfies \(f_i(x)+s_i=0\). Using this transformation we obtain the problem \[\] The new variable \(s_i\) is called slack variable associated with the original inequality constraints.Indeed, if \((x,s)\) is feasible for the above problem, then x is feasible for the original problem. Conversely, if x is feasible for the original problem, then \((x,s)\) is feasible for the above problem. Thant means x is optimal for the original problem if and only if (x,s) is optimal for the above problem.

==Eliminating equality constraints==

If we can explicitly parametrize all solutions of the equality constraints \[ h_i(x)=0,\quad i=1,\dotsm,p \] using some parameters \(z\in R^k\). That means suppose the function \(\phi:R^k\rightarrow R^n\) is such that \(x\) satisfies the above if and only if there is some \(z\in R^k\) such that \(x=\phi(z)\) . The optimization problem \[\] is equaivalent to the original problem. If z is optimal for the transformed problem, then \(x=\phi(z)\) is optimal for the original problem. Conversely, if x is optimal for the original problem, then there is at least one z such that \(x=\phi(z)\) which is optimal for the transformed problem.

4.2 Convex Optimization

4.2.1 Convex optimization problems in standard form

A convex optimization problem is one of the form \[ \begin{array}{lll} \text{minimize}\quad & \begin{array}{lll} f_0(x) \end{array}\\\hline \text{subject to}\quad & \begin{array}{lll} f_i(x)\le 0 & i=1,\dotsm,m\\ a_i^Tx=b_i & i=1,\dotsm,p \end{array} \end{array} \] where \(f_0,\dotsm,f_m\) are convex functions. There are three additional requirements,

  1. The objective function must be convex
  2. The inequality constraint functions must be convex
  3. The equality constraint functions \(h_i(x)=a_i^Tx-b_i\) must be affine.

We immediately note a important property: The feasible set of a convex optimization problem is convex since it is the intersection of the domain of the problem \[ \mathcal D=\bigcap_{i=0}^m dom f_{i} \] which is a convex set with m sublevel sets \(\{x| f_i(x)\le 0\}\) and p hyperplanes \(\{x|a_i^T x=b_i\}\)

If \(f_0\) is quasiconvex instead of convex, we say the above problem is quasiconvex optimization problem.

If the objective function is strictly convex, then the optimal set contains at most one point.

==Concave maximization problems== \[\] ==Abstract form convex optimization problem==

This book consider the convex problems without the standard form (Abstract convex optimization problem) as not convex problems.

4.2.2 Local and global optima

A fundamental property of convex optimization problem is that any locally optimal point is also (globally) optimal.

4.2.3 An optimality criterion for differentiable \(f_0\)

Suppose that the objective \(f_0\) in a convex optimization problem is differentiable, so that for all \(x,y \in dom f_0\), \[ f_0(y)\ge f_0(x)+\nabla f_0(x)^T(y-x) \] Let X denote the feasible set \[ X=\{x|f_i(x)\le0,i=1,\dotsm,m,h_i(x)=0,i=1,\dotsm,p\} \] Then x is optimal if and only if \(x\in X\) and \[ \nabla f_0(x)^T(y-x)\ge 0 \] for all \(y \in X\)

For unconstrained problems, \[ \nabla f_0(x)=0 \] ==Problems with equality constraints only== \[ \begin{array}{lll} \text{minimize}\quad & \begin{array}{lll} f_0(x) \end{array}\\\hline \text{subject to}\quad & \begin{array}{lll} Ax=b \end{array} \end{array} \] For the convex optimal, for the optimal point \(x\), we have \[ \nabla f_0(x)^T(y-x)\ge 0 \] for all \(y\) satisfying \(Ay=b\) and \(Ax=b\). Then we have \(y=x+v\) and \(v\in \mathcal N(A)\quad(Av=0)\).

Then the problem becomes \[ \nabla f_0(x)^Tv\ge0\quad \forall v\in \mathcal N(A) \] If a linear function is nonnegative on a subspace, then it must be zero on the subspace. That means \[ \nabla f_0(x)^Tv=0\quad \forall v\in \mathcal N(A)\\ \nabla f_0(x)\perp \mathcal N(A)\\ \mathcal R(A^T)\perp\mathcal N(A)\\ \nabla f_0(x) \in \mathcal R(A^T) \] Therefore, there exists \(v\in R^P\) such that \[ \nabla f_0(x)+A^Tv=0 \] Together with the constraints.

There exists a \(v\in R^p\) such that \[ \nabla f_0(x)+A^Tv=0\\ Ax=b \] This is actually the classical Lagrange multiplier optimality condition.

==Minimization over the nonnegative orthant==

We consider the problem \[ \begin{array}{lll} \text{minimize}\quad & \begin{array}{lll} f_0(x) \end{array}\\\hline \text{subject to}\quad & \begin{array}{lll} x\succeq 0 \end{array} \end{array} \] We have optimality condition \[ x\succeq0 \quad \nabla f_0(x)^T(y-x)\ge 0\quad \forall y\succeq 0 \] We see the linear function of y \(\nabla f_0(x)^Ty-\nabla f_0(x)^T x\) is greater than or equal to zero for all y only when \[ \nabla f_0(x)\succeq 0\\ -\nabla f_0(x)^Tx\ge 0 \] But \(x\succeq 0\) and \(\nabla f_0(x)\succeq 0\), and therefore \(\nabla f_0(x)^Tx=0\)

The optimality condition can be expressed as \[ x\succeq 0\quad \nabla f_0(x)\succeq 0\quad x_i(\nabla f_0(x))_i=0 \space(\text{complementary condition}) \]

4.2.5 Quasiconvex Optimization

Recall that a quasiconvex optimization problem has the standard form \[ \begin{array}{lll} \text{minimize}\quad & \begin{array}{lll} f_0(x) \end{array}\\\hline \text{subject to}\quad & \begin{array}{lll} f_i(x)\le 0 & i=1,\dotsm,m\\ Ax=b \end{array} \end{array} \] where \(f_0\) is quasiconvex and \(f_i\) are convex.

==Locally optimal solutions and optimality conditions==

The most important difference between the convex and quasiconvex optimization is that quasiconvex optimization problems can have locally optimal solutions.

It follows from the first order condition for quasiconvex optimization problem that x is optimal if \[ x\in X,\quad \nabla f_0(x)^T(y-x)>0\text{ for all }y\in X /\{x\} \] There are two differences between this criterion and the analogous one for convex optimization.

  1. This condition is only sufficient for optimality.
  2. This condition requires the gradient of \(f_0\) to be nonzero.

==Quasiconvex optimization via convex feasibility problems==

One general approach to quasiconvex optimization relies on the representation of the sublevel sets of a quasiconvex function via a family of convex inequalities. \[ f_0(x)\le t \iff \phi_t(x)\le 0 \] Let \(p^\star\) denote the optimal value of the quasiconvex function. If the feasibility problem \[ \begin{array}{lll} \text{minimize}\quad & \begin{array}{lll} x \end{array}\\\hline \text{subject to}\quad & \begin{array}{lll} \phi_t(x)\le 0\\f_i(x)\le 0\\Ax=b \end{array} \end{array} \] is feasible. Then \(p^\star \le t\)
