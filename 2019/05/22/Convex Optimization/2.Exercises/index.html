2.2

Prove sufficiency,

if a set is convex, then its intersection with any line is convex.

The intersection of two convex set is convex. Proved.

Prove necessity,

if the intersection of a set and any line is convex, the set is convex.

Take any two points from the set. The line through the two points is convex. Therefore, the set is convex.

2.10

A set is convex if and only if its intersection with an arbitrary line is convex

Define the arbitrary line \(\{\hat x+tv|t\in R\}\)

Then the intersection set is \[ \{\hat x+tv|(\hat x+tv)^TA(\hat x+tv)+b^T(\hat x+tv)+c\le 0,t\in R\}\\ \{\hat x+tv|(v^TAv )t^2+(2\hat x^TAv+b^Tv)t+\hat x^T A\hat x+b^T\hat x+c\le 0,t\in R\} \] The intersection set needs to be convex for every \(\hat x\) and \(v\) . That mean \(v^TAv\ge0\quad \forall v \implies A\succeq 0\)
