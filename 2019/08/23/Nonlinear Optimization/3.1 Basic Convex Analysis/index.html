Basic Convex Analysis

Linear functions are very important in linear programming and applied mathematics. However, convex functions have a wider range of applications than the class of linear functions. In fact, linear functions are special convex functions.

3.1 Convex sets

Definition 3.1 (Convex set)

A set \(D\subseteq R^n\) is said to be convex if for any two points \(x,y\in D\), the linear segment joining \(x\) and \(y\) also lies in \(D\). That is, \[ x,y\in D\implies \lambda x+(1-\lambda )y \in D \quad\forall \lambda \in[0,1]. \] Notation

We will sometimes use \([x,y]\) to denote the line segment joining \(x\) and \(y\), i.e. \([x,y]=\{\lambda x+(1-\lambda)y:\lambda\in[0,1]\}\).

To prove thee a given set \(C\) is convex, we use the following procedures,

  1. Choose two arbitrary points \(x,y\in C\). Choose any \(\lambda \in [0,1]\)
  2. Argue that \(\lambda x+(1-\lambda )y\in C\).

Example (Half space)

Given \(a\in R^n\), the half space \(H=\{x\in R^n|a^Tx\le \alpha\}\) is a convex set.

Proof.

Consider \(x,y\in H\), and choose \(\lambda \in [0,1]\), we have \[ a^T(\lambda x+(1-\lambda )y)=\lambda a^Tx+(1-\lambda)a^Ty\le \lambda \alpha+(1-\lambda)\alpha=\alpha \] Example (n-ball)

Given \(a\in R^n\), \(\alpha >0\). The n-ball \[ \bar B(a,\alpha)=\{x\in R^n\,|\,||x-a||\le \alpha\} \] is a convex set.

Consider \(x,y\in \bar B\), and choose \(\lambda \in [0,1]\), we have \[ ||\lambda x+(1-\lambda )y-a||=||\lambda x+(1-\lambda)y-\lambda a-(1-\lambda)a||\\ =||\lambda(x-a)+(1-\lambda)(y-a)||\le \lambda ||x-a||+(1-\lambda )||y-a||\\ \le \lambda \alpha +(1-\lambda)\alpha =\alpha \] Proposition 3.1

(Finite or Infinite) intersection of convex sets is a convex set.

Proof.

We only need to prove intersection of two convex sets are convex.

Choose \(x,y \in \bigcap_{i=1}^k C_i\) and \(\lambda\in [0,1]\). Since \(x,y \in C_i\) and \(C_i\) is a convex set. We have \(\lambda x+(1-\lambda)y\in C_i\;\forall i=1,\dotsm k\). That means \(\lambda x+(1-\lambda)y\in \bigcap_{i=1}^k C_i\). Q.E.D.

Example (Polyhedron or polyhedral set)

The polyhedron or polyhedral set are convex set. \(\{Ax\le b\}\)

3.2 Convex and concave functions

Definition 3.2 (Convex function/Strictly convex function)

Let \(D\subseteq R^n\) be a convex set. Consider a function \(f:D\rightarrow R\).

  1. The function is said to be convex if \[ f(\lambda x+(1-\lambda )y)\le \lambda f(x)+(1-\lambda )f(y)\quad \forall x,y\in D,\lambda\in[0,1] \]
  2. The function is said to be strictly convex if \[ f(\lambda x+(1-\lambda )y)< \lambda f(x)+(1-\lambda )f(y)\quad \forall x,y\in D,\lambda\in[0,1] \] Notation

\(-f\) is convex $$ \(f\) is concave.

To prove that a given function \(f:D\rightarrow R\) is convex, we use the following procedures,

  1. Choose two arbitrary points \(x,y\in D\). Choose any \(\lambda \in [0,1]\)
  2. Argue that \(f(\lambda x+(1-\lambda )y)\le \lambda f(x)+(1-\lambda)f(y)\).

Example Let \(f:R^2\rightarrow R\) where \(f(x)=||x||=(x_1^2+x_2^2)\frac{1}{2}\), show that \(f\) is convex in R.

Choose \(x,y\in R^2\), and \(\lambda \in [0,1]\), then we have \[ f(\lambda x+(1-\lambda)y)=||\lambda x+(1-\lambda )y||\le \lambda ||x||+(1-\lambda)||y|| \] Example Let \(a\in R^n\) and \(\alpha \in R\). Consider \(f:R^n\rightarrow R,f(x)=a^Tx+\alpha\)

Choose \(x,y\in R^2\), and \(\lambda \in [0,1]\), then we have \[ f(\lambda x+(1-\lambda)y)=a^T(\lambda x+(1-\lambda)y)+\alpha =\lambda f(x)+(1-\lambda)f(y) \] Example

\(f:R^2\rightarrow R\) defined by \(f(x)=x_1^2-4x_2\) is convex

Choose \(x,y\in R\), and \(\lambda \in [0,1]\), then we have \[ f(\lambda x+(1-\lambda)y)=(\lambda x_1+(1-\lambda)y_1)^2-4(\lambda x_2+(1-\lambda)y_2)\\ =\lambda^2x_1^2+2\lambda x_1(1-\lambda)y_1+(1-\lambda)^2y_1^2-4\lambda x_2-4(1-\lambda)y_2\\ \lambda f(x)+(1-\lambda)f(y)=\lambda x_1^2-4\lambda x_2+(1-\lambda)x_1^2-4(1-\lambda)x_2 \] Do subtraction. Then Q.E.D.

Proposition 3.2 If \(f_1,f_2:D\rightarrow R\) are convex function on a convex set \(D\subseteq R^n\), then

  1. \(f_1+f_2\) is convex
  2. \(\alpha f_1\) is convex for \(\alpha \ge 0\)

Proof.

Let \(g=f_1+f_2\). For \(x,y\in D\) and \(\lambda \in [0,1]\), we have \[ g(\lambda x+(1-\lambda)y)=f_1(\lambda x+(1-\lambda)y)+f_2(\lambda x+(1-\lambda)y)\\ \le \lambda g+(1-\lambda)g \] Q.E.D for 1.

Straightforward for 2.

Corollary 3.1 Let \(f_1,f_2,\dotsm,f_n:D\rightarrow R\) are convex functions on a convex set \(D\subseteq R^n\). Then \[ f(x)=\sum_{j=1}^k\alpha_jf_j(x) \] where \(\alpha_j\ge 0\) is also convex function. If one of the functions is strictly convex, then \(f(x)\) is strictly convex.

Proof. It is straightforward to use the previous proposition.

Proposition 3.3 Let \(h:D\rightarrow R\) be convex function and \(g:\mathcal X\rightarrow R\) be a nondecreasing convex function with \(h(D)\subset \mathcal X\). Then the composition function \(f=g\circ h:D\rightarrow R\) is a convex function.

Proof. \[ f(\lambda x+(1-\lambda )y)=g(h(\lambda x+(1-\lambda)y))\\ \le g(\lambda h(x)+(1-\lambda)h(y))\text{ (Since $h$ is convex and $g$ is non-decreasing)}\\ \le \lambda g(h(x))+(1-\lambda)g(h(y)) \text{ (Since $g$ is convex)} \] Q.E.D.

Proposition 3.4 Let \(h:D\rightarrow R\) be convex function and \(g:\mathcal X\rightarrow R\) be a nonincreasing concave function with \(h(D)\subset \mathcal X\). Then the composition function \(f=g\circ h:D\rightarrow R\) is a concave function.

Proof. Same as the previous proposition.

The next result depicts a useful relationship between convex function and convex set.

Proposition 3.5 Suppose \(D\subset R^n\) is convex. If \(f:D\rightarrow R\) is convex, then for any \(\alpha \in R\), the set \[ S_\alpha=\{x\in D\;|\;f(x)\le \alpha\} \] is convex.

Proof.

We choose \(x,y\in S_\alpha\), and choose \(\lambda \in [0,1]\). Then we have \[ f(\lambda x+(1-\lambda)y)\le \lambda f(x)+(1-\lambda )f(y)\le \lambda \alpha +(1-\lambda )\alpha = \alpha \] That means the point \(\lambda x+(1-\lambda)y\) is also in the set. Q.E.D.

Proposition 3.6 Suppose \(f:D\rightarrow R\) is a function defined on a convex set \(D\subset R^n\). The epigraph of \(f\) is the following subset of \(R^{n+1}\): \[ E_f=\{[x;\alpha]\;|\; x\in D,\alpha\in R,f(x)\le \alpha\} \] The epigraph \(E_f\) is a convex set if and only if \(f\) is convex function.

Proof.

\(\implies\)

If the epigraph \(E_f\) is a convex set, then we choose \(x,y\in E_f\), and choose \(\lambda \in [0,1]\). We have \[ \lambda [x;\alpha]+(1-\lambda)[y;\beta]\in E_f \] for any \(\alpha,\beta \in R\).

Then we have \[ [\lambda x+(1-\lambda )y;\lambda f(x)+(1-\lambda )f(y)]\in E_f \] Then by definition, we have \(f(\lambda x+(1-\lambda )y)\le \lambda f(x)+(1-\lambda )f(y)\). Q.E.D.

\(\Longleftarrow\)

If the function \(f\) is a convex function, then we choose \([x;\alpha],[y;\beta]\in E_f\), and choose \(\lambda \in [0,1]\). We have \[ \lambda [x;\alpha]+(1-\lambda )[y;\beta]=[\lambda x+(1-\lambda)y;\lambda \alpha+(1-\lambda)\beta] \] Then we find \[ f(\lambda x+(1-\lambda)y)\le \lambda f(x)+(1-\lambda)f(y)\le \lambda \alpha+(1-\lambda)\beta \] That mean \[ [\lambda x+(1-\lambda)y;\lambda \alpha+(1-\lambda)\beta]\in E_f \] Q.E.D.

Proposition 3.7 (Jensen's inequality)

Suppose \(f:S\rightarrow R\) is a convex function on a convex set \(S\subset R^n\), and \(x_1,x_2,\dotsm,x_n\in S\). Let \[ x=\sum_{j=1}^k \lambda_j x_j,\quad \sum_{j=1}^k\lambda_j=1,\quad \lambda_j\ge 0 \] Then \[ f(x)\le \sum_{j=1}^k\lambda_k f(x_k) \]

3.3 Differentiable convex functions

Testing or checking the convexity of a function by definition is not a easy task. However, when the function is twice differentiable, the task becomes much easier.

Theorem 3.1 (Tangent plane characterization)

Suppose that \(f(x)\) has continuous first partial derivatives on an open convex set \(S\) in \(R^n\). Then,

  1. the function is convex if and only if

\[ f(x)+\nabla f(x)(y-x)\le f(y)\quad \forall x,y\in S \]

  1. the function is strictly convex if and only if

\[ f(x)+\nabla f(x)(y-x)< f(y)\quad \forall x,y\in S \]

Proof.

\(\implies\)

Suppose \(f:D\rightarrow R\) is convex, then we have \[ f(\lambda y+(1-\lambda )x)\le \lambda f(y)+(1-\lambda )f(x)\quad \forall x,y\in D,\forall \lambda \in [0,1] \] Then we have \[ \frac{f(x +\lambda (y-x))-f(x)}{\lambda}\le f(y)-f(x) \] When \(\lambda \rightarrow 0\), the left hand side is \(\nabla f(x) ^T(y-x)\)

\(\Longleftarrow\)

If we have \(f(x)+\nabla f(x)(y-x)\le f(y)\)

Let \(u,v\in S\) and \(\lambda \in [0,1]\), we consider \(w=u+(1-\lambda )v\)

We have \[ f(w)+\nabla f(w)(u-w)\le f(u)\\ f(w)+\nabla f(w)(v-w)\le f(v) \] Since we have \(v-w=-\frac{\lambda}{1-\lambda}(u-w)\), we can cancel \(\nabla f(\cdot)\). Q.E.D.

Remark The equation \(z=f(x)+\nabla f(x)^T(y-x)\) define the tangent plane to the surface \(z=f(x)\) at \(y=x\). Thus the theorem shows that the tangent plane is below the convex function.

Proposition 3.7 (Monotone gradient condition)

Let \(f:S\rightarrow R\) be differentiable function on the open convex subset \(S\) of \(R^n\). Then \(f\) is convex on \(S\) if and only if \[ \left\langle \nabla f(x)-\nabla f(y),x-y\right\rangle\ge 0\quad \forall x,y\in S \] Proof.

\(\implies\) \[ f(x)+\nabla f(x)^T(y-x)\le f(y)\\ f(y)-\nabla f(y)^T(y-x)\le f(x) \] By doing the summation, Q.E.D.

\(\Longleftarrow\)

Suppose we have \(\left\langle \nabla f(x)-\nabla f(y),x-y\right\rangle\ge 0\quad \forall x,y\in S\). Define \(g(t)=f(x+t(y-x))\). Then we have \[ g'(t)=<\nabla f(x+t(y-x)),y-x> \] Then we have \[ g'(t)-g'(0)=<\nabla f(x+t(y-x))-\nabla f(x),t(y-x)>\ge 0\quad \forall t\in[0,1] \] Then we have \[ g(1)=f(y)=g(0)+\int _0^1 g'(t)dt\ge g(0)+g'(0)=f(x)+<\nabla f(x),y-x> \] Q.E.D.

Theorem 3.2 (Test for convexity of a differentiable function)

Suppose that \(f(x)\) has continuous second partial derivatives on an open convex set \(D\subset \mathbb R^n\), the function is convex on \(D\) if and only if the Hessian matrix is positive semidefinite.

3.4 Strongly convex functions

Definition 3.3 Let \(D\subseteq \mathbb R^n\) be a convex set. A function \(f:D\rightarrow \mathbb R\) is said to be strongly convex if there exists a constant \(c>0\) such that for all \(x,y\in D\), and \(\lambda \in[0,1]\), \[ f(\lambda x+(1-\lambda)y)\le \lambda f(x)+(1-\lambda)f(y)-\frac{c}{2}\lambda(1-\lambda)||x-y||^2 \] Theorem 3.3 Suppose \(f:D\rightarrow \mathbb R\) is a differentiable function. Then the following statements are equivalent.

  1. \(f(y)\ge f(x)+<\nabla f(x),y-x>+(c/2)||y-x||^2\)

  2. \(g(x)=f(x)-(c/2)||x||^2\) is convex

  3. \(<\nabla f(x)-\nabla f(y),x-y>\ge c||x-y||^2\)

  4. \(f\) is strongly convex.

Proof. (\(1\iff 2\))

\(\Longleftarrow\)

If \(g(x)=f(x)-(c/2)||x||^2\) is convex, then we have \[ g(y)\ge g(x)+\nabla g(x)(y-x)\\ \implies f(y)-(c/2)||y||^2\ge f(x)-(c/2)||x||^2+\nabla f(x)^T(y-x)-cx^T(y-x)\\ \implies f(y)\ge f(x)+\nabla f(x)^T(y-x)+(c/2)\left(||y||^2-2x^Ty+||x||^2\right) \] \(\implies\)

By Theorem 3.1.

(\(2\iff 3\)) Monotone gradient.

(\(1\iff 4\)) Straightforward. Q.E.D.

3.5 Unconstrained convex programming problem

A convex programming problem is a minimization problem whose objective function is convex and the feasible set is convex. In general, a local minimizer is not a global minimizer. However, for a convex minimization problem, a local minimizer is also a global minimizer.

An unconstrained convex nonlinear programming problem is the following: \[ \text{minimize }f(x)\\x\in D. \] Theorem 3.4

Let \(D\) be a nonempty open convex subset of \(\mathbb R^n\), and \(f:D\rightarrow \mathbb R\) is a convex function. Consider the convex minimization problem. Suppose \(x^\star \in D\) is a local minimizer to the problem, then

  1. \(x^\star\) is a global minimizer
  2. If \(f\) is strictly convex, then \(x^\star\) is the unique global minimizer.

Proof.

  1. We assume \(x^\star\in D\) is a local minimizer, which means there exists a ball \(B_\epsilon(x^\star)\), such that for all points \(x\in B_\epsilon(x^\star)\), \(f(x)\ge f(x^\star)\). Then we need to prove \(x^\star\) is a global minimizer, which means for all points \(y\in D\), we have \(f(y)\ge f(x^\star)\).

    We consider the point \(\lambda y+(1-\lambda )x^\star \in D\). If $$ is enough small, then the point is also in the ball. That means we have \[ \lambda f(y)+(1-\lambda)f(x^\star)\ge f(\lambda y+(1-\lambda)x^\star)\ge f(x^\star). \] Q.E.D.

  2. Assume \(x^\star \in D\) is the global optimizer. If there is another global minimizer \(\bar x\), for which we have \(f(\bar x)=f(x^\star)\).

    Then we choose \(w=(1/2)x^\star+(1/2)\bar x\). \[ f(w)=f((1/2)x^\star+(1/2)\bar x)< (1/2)f(x^\star)+(1/2)f(\bar x)=f(x^\star), \] which contradicts that the point \(x^\star\) is the global optimal point. Q.E.D.

Corollary 3.2

If \(f\) is a convex function with continuous first partial derivatives on some open convex set \(D\), then any stationary point of \(f\) is a global minimizer of \(f\).

3.6 Unconstrained convex quadratic programming

Definition (Quadratic function)

A quadratic function \(q:\mathbb R^n\rightarrow \mathbb R\) is defined by \[ q(x)=\frac{1}{2}x^TQx+c^Tx \] The quadratic function is a convex function if \(Q\) is positive semidefinite.

The point \(x^\star \in S\) is a global minimizer of the problem if and only if \(Qx^\star =-c\).

3.8 Convex Separation

Definition 3.5

  1. Affine hull \(\text{aff(S)}\) of a set S in \(\mathcal E\) is the smallest affine set containing \(S\), i.e. in intersection of all affine set containing \(S\).

\[ \text{aff(S)}=\{\sum_{i=1}^k\alpha_ix_i\;|\; \sum_{i=1}^k \alpha_i=1,x_i\in S,k>0\} \]

  1. Convex hull \(\text{conv(S)}\) of a set S in \(\mathcal E\) is the smallest convex set containing \(S\), i.e. in intersection of all convex set containing \(S\).

\[ \text{conv(S)}=\{\sum_{i=1}^k\alpha_ix_i\;|\; \sum_{i=1}^k \alpha_i=1,x_i\in S,\alpha_i\ge 0,k>0\} \]

  1. A point \(x\) is a relative interior point of \(S\) if \(x\) is an interior point relative to \(\text{aff(S)}\). The set of all relative interior points of \(S\) is denoted as \(\text{ri(S)}\)

\[ \text{ri(S)}=\{x\in \text{aff(S)}\;|\; \exist \varepsilon >0, B_{\varepsilon}(x)\cap \text{aff(S)}\subset S\} \]

Proposition 3.9 Let \(C\) be a subset of a Euclidean space \(\mathcal E\). If \(C\) is convex, then \(\text{aff(ri C)}=\text{aff(C)}=\text{aff(cl(C))}\)

In the next proposition we will prove that any point outside a closed convex set can be separated from that set with a hyperplane.

The following assertions hold for affine.

  1. A set \(\Omega\subset R\) is affine if and only if \(\Omega\) contains all affine combinations of its elements.
  2. If \(\Omega_1\) is an affine subset of \(R^n\) and \(\Omega_2\) is an affine subsets of \(R^m\), then \(\Omega_1\times \Omega_2\) is an affine subset of \(R^n\times R^m\)
  3. Let \(B:R^n\rightarrow R^m\) be an affine mapping if \(\Omega\) is an affine subset of \(R^n\) and \(\Theta\) is an affine subset of \(R^m\). Then the image \(B(\Omega)\) is an affine subset of \(R^m\) and the inverse image \(B^{-1}(\Theta)\) is an affine subset of \(R^n\).
  4. Let \(\Omega,\Omega_1,\Omega_2\) are affine subsets of \(R^n\). Then the sum \(\Omega_1+\Omega_2\) and the scalar product \(\lambda \Omega\) for any \(\lambda \in R\) are also affine subsets of \(R^n\).
  5. Given \(\Omega\subset R^n\), its affine hull is the smallest affine set containing \(\Omega\).
  6. A set \(\Omega\) is a linear subspace of \(R^n\) if and only if \(\Omega\) is an affine set containing the origin.

Definition (Affine independence)

The elements \(v_0,\dotsm,v_m\) in \(R^m\), \(m\ge 1\) are affinely independent if \[ \sum_{i=1}^m \lambda_i v_i=0,\quad \sum_{i=1}^m \lambda_i=0\implies \lambda_i=0\quad \forall i=0,\dotsm,m \] Roughly speaking, affine independence is like linear independence but without the restriction that the subset of lower dimension the points lie in contains the origin. So three points in space are affinely independent if the smallest flat thing containing them is a plane. They're affinely dependent if they lie on a line (or are the same point).

Definition (Simplex)

Let \(v_0,\dotsm,v_m\) be affinely independent in \(R^n\). Then the set \[ \Delta_m:conv\{v_i\;|\; i=0,\dotsm,m\} \] is called an m-simplex in \(R^n\) with the vertices \(v_0,\dotsm,v_m\)

Let \(\Omega\) be a nonempty, convex set in \(R^n\) of dimension \(m\ge 1\). Then there exists \(m+1\) affinely independent elements \(v_0,\dotsm,v_m\) in \(\Omega\).
