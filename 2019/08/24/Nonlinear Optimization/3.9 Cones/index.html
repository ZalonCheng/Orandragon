3.9 Cones

==Definition==

Consider \(p:\mathcal E\rightarrow (-\infty,\infty]\).

  1. \(p\) is said to be a proper function if \(p(x)\) is not identically equal to \(\infty\).
  2. \(p\) is said to be closed if its epi-graph is a closed subset of \(R\times \mathcal E\).
  3. The domain of \(p\) is defined to be set

\[ \text{dom} (p)=\{x\in R^n|f(x)<\infty\} \]

For a nonempty closed set \(C\subset R^n\), \(\delta_C\) is a proper closed function.

==Definition==

  1. (Linearity space) Let \(C\subset R^n\) be a closed convex cone. The linearity space of \(C\) is the set

\[ \text{lin} (C)=C\cap(-C) \]

It is the largest linear subspace of \(\mathcal E\) contained in C.

  1. (Dual and polar cone) The dual of any cone \(S\) (not necessarily convex) is defined by

\[ S^\star =\{y\in\mathcal E|<x,y>\ge 0\;\forall x\in S\} \]

​ The polar cone of \(S\) is \(S^o=-S^\star\)

  1. (self-dual) Let C be a cone in \(\mathcal E\). If \(C^\star=C\), then \(C\) is said to be self-dual.

==Proposition==

Let \(C\) be a cone in \(\mathcal E\).

  1. \(C^\star\) is a closed convex cone.
  2. If \(C\) is a nonempty closed convex cone, then \((C^\star)^\star=C\).

3.10 Normal cones

==Definition== (Normal cone)

Let \(C\subset \mathcal E\) be a convex set and \(\bar x\in C\). The normal cone of \(C\) at \(\bar x\in C\) is defined by \[ N_c(\bar x)=\{z\in \mathcal E|<z,x-\bar x>\le 0\quad \forall x\in C\} \] By convention, we let \(N_C(\bar x)=\empty\) if \(\bar x\notin C\).

==Proposition== Let \(C\subset \mathcal E\) be a convex set and \(\bar \in C\). Then

  1. \(N_C(\bar x)\) is a closed convex cone
  2. If \(\bar x\in int (C)\), then \(N_c(\bar x)=\{0\}\).
  3. If \(C\) is also a cone, then \(N_c(\bar x)\subset C^o\)

==Proposition==

Let \(C\subset R^n\) be a closed convex set. Then for any \(u,y\in C\), \[ u\in N_c(y)\iff y=\Pi_C(y+u) \] ==Proposition==

Let \(C_1,C_2 \subset R^n\) be convex sets such that \(\bar x\in C_1\cap C_2\).

  1. It holds that \(N_{c_1}(\bar x)+N_{c_2}(\bar x)\subset N_{C_1\cap C_2}(\bar x)\)
  2. If the following relative interior condition holds

\[ ri(C_1)\cap ri(C_2)\neq \empty \]

​ then we also have \[ N_{C_1\cap C_2}(\bar x)\subset N_{c_1}(\bar x)+N_{c_2}(\bar x) \]
