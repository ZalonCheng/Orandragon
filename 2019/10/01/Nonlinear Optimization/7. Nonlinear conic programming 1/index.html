7. Nonlinear Conic Programming

We consider the following optimization problem, \[ \text{(OP)}\quad \begin{array}{rCl} \min & f(x)\\ \text{subject to}& g(x)=0\\ &h(x)\in \mathcal C, \end{array} \] where \(f:\mathcal X\rightarrow \mathbb R,g:\mathcal X\rightarrow \mathcal U\), and \(h:\mathcal X\rightarrow \mathcal V\) are continuously differentiable, \(\mathcal X,\mathcal U\) and \(\mathcal V\) are finite dimensional Euclidean spaces each equipped with a scalar product \(\langle \cdot,\cdot\rangle\) and its 0induce norm \(||\cdot||\), and \(\mathcal C\subseteq \mathcal V\) is a closed convex set in \(\mathcal V\).

Let \(\mathcal Y=\mathcal U\times \mathcal V\), \(\mathcal K=\{0^\mathcal U\}\times \mathcal C\subset \mathcal Y\). Define \(G:\mathcal X\rightarrow \mathcal Y\) by \[ G(X)=(g(X),h(X)),\quad X\in\mathcal X. \] The problem can be written as the following compact form, \[ \text{(COP)}\quad \begin{array}{rCl} \min & f(X)\\ \text{subject to}& G(X)\in\mathcal K. \end{array} \] Define the Lagrangian function \(L:\mathcal X\times \mathcal Y\rightarrow \mathbb R\) for (COP) by \[ L(X,\mu)=f(X)-\langle \mu, G(X)\rangle,\quad X\in\mathcal X,\mu\in\mathcal Y. \] There is no constraint on \(\mu\) for the general Lagrangian function. Therefore, \(\mathcal Y\) here, we will give explanation in the later.

The Lagrangian dual function is defined by \[ \theta(\mu)=\inf\left\{L(X,\mu )\;|\; X\in\mathcal X\right\}. \] The dual problem is \[ \max \{\theta(\mu)\;|\; \mu \in\mathcal K^*\}, \] where \(\mathcal K^* =\mathcal U\times \mathcal C^*\) is the dual cone of \(\mathcal K\). Note that \((\{0^\mathcal U\})^*=\mathcal U\) and \(\mathbb S_+^n\) is self-dual.

We can see in the conic programming, we only consider \(\mu\in \mathcal K^*\) in the dual problem. However, if needed, we can also put this constraint to the Lagrangian dual function and Lagrangian function like what we usually did in basic nonlinear programming.

We also define the Lagrangian function in the conic programming as \(L(x,\mu)=f(x)\textbf{ minus } \langle \mu,G(x)\rangle\) instead of plus. This is for convenience because the cone is usually defined as “positive”, which means \(h(x)\succeq 0\) in the conic programming instead of \(h(x)\le 0\) in the basic nonlinear programming is the common case.

Example

If we have \(\mathcal X=\mathbb R^n\), \(\mathcal U=\mathbb R^m\), \(\mathcal V=\mathbb R^q\) and \(\mathcal C=\mathbb R^q_+\). In this case, the problem reduces to the following, \[ \text{(NLP)}\quad \begin{array}{rCl} \min & f(x)\\ \text{subject to}& g(x)=0\\ &h(x)\ge 0. \end{array} \] If we denote \(G(x)=(g(x),h(x))\), \(\mathcal K=\{0^m\}\times \mathbb R^n_+\), there exists \(\mu\in\mathbb R^m\times \mathbb R^n\), such that the Lagrange function is defined as \[ L(x,\mu)=f(x)-\langle \mu,G(x)\rangle. \] The Lagrange dual function is defined as \[ \theta(\mu)=\min_x L(x,\mu)=\min_x\left(f(x)-\langle\mu,G(x)\rangle\right),\quad \mu\in\mathbb R^m\times \mathbb R^n. \] Since \(\left(\mathbb R^n_+\right)^*=\mathbb R^n_+\), the Lagrange dual problem is defined as \[ \text{(D)}\quad \begin{array}{rCl} \min & \theta (\mu)\\ \text{subject to}& \mu\in\mathcal K^*=\mathbb R^n\times \mathbb R^n_+. \end{array} \] Example (semidefinite linear programming)

If \(\mathcal X=\mathbb S^n\), \(\mathcal U=\mathbb R^m\), \(\mathcal V=\mathbb S^n\), and \(f(X)=\langle C,X\rangle,\) \(g(X)=\mathcal A(X)-b\), \(h(X)=X\) and \(\mathcal C=\mathbb S_+^n\), then the problem (OP) reduces to the following linear semidefinite programming problem, \[ \text{(SDP)}\quad \begin{array}{rCl} \min & \langle C,X\rangle\\ \text{subject to}& \mathcal A(X)-b=0\\ &X\succeq 0(X\in\mathbb S^n_+), \end{array} \] where \(C\in \mathbb S^n\), \(\mathcal A:\mathbb S^n\rightarrow \mathbb R^m\) is a linear mapping defined by \[ \mathcal A(X)= \begin{bmatrix} \langle A_1,X\rangle \\ \vdots \\ \langle A_m,X\rangle \end{bmatrix},\quad X\in\mathbb S^n, \] where \(A_k\in \mathbb S^n,k=1,\dotsm,m\), are the constraints matrices. Let \(\mathcal A^*:\mathbb R^m\rightarrow \mathbb S^n\) be the adjoint of \(\mathcal A\) defined by \[ \mathcal A^*y=\sum_{k=1}^m y_kA_k,\quad y\in \mathbb R^m. \] This can be shown by the definition of the adjoint of operator. \[ \begin{array}{rcl} \langle \mathcal A^* y,X\rangle &=&\langle y,\mathcal A X\rangle\\ &=&\displaystyle\sum_{i=1}^m y_i\langle A_i,X\rangle \\ &=&\displaystyle\sum_{i=1}^m y_i\sum_{j=1}^ne_j^TA_iXe_j, \\ \mathbf 1^T\mathcal A^*yX \mathbf 1 &=& \displaystyle\sum_{i=1}^m y_i \mathbf 1^TA_iX\mathbf 1\\ &=& \mathbf 1^T\displaystyle\sum_{i=1}^m y_i A_iX\mathbf 1, \end{array} \] Therefore, we have \(\mathcal A^*y=\sum_{i=1}^m y_iA_i\).

We have the Lagrange function, \[ \begin{array}{rcl} L(X,y,Z)&=&\langle C,X\rangle +\langle y,b-\mathcal A(X) \rangle+\langle Z,-X\rangle\\ &=&\langle y,b\rangle+\langle C-\mathcal A^*y-Z,X\rangle\quad X\in\mathbb S^n,y\in\mathbb R^n,Z\in\mathbb S^n. \end{array} \] Then \[ \begin{array}{rcl} \theta(y,Z)&=&\displaystyle \inf_{X\in\mathbb S^n} \left\{\langle y,b\rangle+\langle C-\mathcal A^*y-Z,X\rangle\right\}\\ &=&\displaystyle\inf_{X\in\mathbb S^n} \left\{\langle y,b\rangle+\langle C-\mathcal A^*y-Z,X\rangle\right\}\\ &=& \begin{cases} \langle y,b\rangle&\text{if }C-\mathcal A^*y-Z=0\\ -\infty & \text{otherwise.} \end{cases} \end{array} \] Therefore, the dual problem is given by, \[ \text{(D)}\quad \begin{array}{rCl} \min & \langle y,b\rangle\\ \text{subject to}& C-\mathcal A^*y-Z=0\\ &Z\succeq 0. \end{array} \] Also, we can consider the compact form. If we define \(G(X)=(\mathcal A(X)-b, X)\), \(\mathcal K=\{0^m\}\times \mathbb S^n_+\), then we have \[ \text{(SDP)}\quad \begin{array}{rCl} \min & f(X)=\langle C,X\rangle\\ \text{subject to}& G(X)\in\mathcal K. \end{array} \] Then there exists \(\mu=(y,Z)\in\mathbb R^n\times \mathbb S^n\), such that the Lagrange function is defined as \[ \begin{array}{rcl} L(X,\mu)=L(X,y,Z)&=&f(X)-\langle \mu, G(X)\rangle\\ &=&\langle C,X\rangle-\langle y,\mathcal A(X)-b\rangle-\langle Z,X\rangle\\ &=&\langle C,X\rangle-\langle \mathcal A^*(y), X\rangle+\langle y,b\rangle-\langle Z,X\rangle\\ &=&\langle C-\mathcal A^*(y)-Z,X\rangle+\langle y,b\rangle\\ \end{array} \] The we have the Lagrange dual function, \[ \begin{array}{rcl} \theta(y,Z) &=& \begin{cases} \langle y,b\rangle&\text{if }C-\mathcal A^*y-Z=0\\ -\infty & \text{otherwise.} \end{cases} \end{array} \] Then the Lagrange dual problem is \[ \text{(D)}\quad \begin{array}{rCl} \max & \theta(y,Z)\\ \text{subject to}& (y,Z)\in\mathcal K^*=\mathbb R^n\times \mathbb S^n_+, \end{array} \] that is \[ \text{(D)}\quad \begin{array}{rCl} \max & \langle y,b\rangle\\ \text{subject to}& Z\in\mathbb S^n_+\\ &C-\mathcal A^*y-Z=0. \end{array} \] Example

Let \(G\in\mathbb R^{p\times n}\) and \(H\in\mathbb R^{q\times n}\) for the linear map \(\mathcal A:\mathbb S^n\rightarrow \mathbb R^{p\times q}\) defined by \(\mathcal A(X)=GXH^T\). Show that the adjoint map \(\mathcal A^*:\mathbb R^{p\times q} \rightarrow \mathbb S^n\) is given by \(\mathcal A^*(Y)=\frac{1}{2}\left(G^TYH+H^TY^TG\right)\).

Proof. \[ \begin{array}{rcl} \langle \mathcal A(X),Y\rangle&=& \langle GXH^T,Y\rangle\\ &=& \text{tr}(HXG^TY)\\ &=& \text{tr}(XG^TYH)\\ &=& \frac{1}{2}\text{tr}\left(X(G^TYH+H^TY^TG)\right)\\ &=&\langle X,\frac{1}{2}(G^TYH+H^TY^TG)\rangle. \end{array} \] Therefore, we have \(\mathcal A^*(Y)=\frac{1}{2}\left(G^TYH+H^TY^TG\right)\). (ATTENTION: \(\mathcal A^*:\mathbb R^{p\times q}\rightarrow \mathbb S^n\))

Example (semidefinite programming least squares problem)

If \(f(x)=\frac{1}{2}||X-B||^2\) where \(B\in \mathbb S^n\) is a given matrix. \[ \text{(SDPLS)}\quad \begin{array}{rCl} \min & \frac{1}{2}||X-B||^2\\ \text{subject to}& \mathcal A(X)=b\\ &X\succeq 0. \end{array} \] In particular, when \(\mathcal A(X)=\text{diag} (X)\) and \(b=e\), we get the nearest correlation matrix problem.

Example (dual of semidefinite programming least squares problem)

The problem can be denoted as \[ \text{(SDPLS)}\quad \begin{array}{rCl} \min & \frac{1}{2}||X-B||^2\\ \text{subject to}& G(X)\in \mathcal K, \end{array} \] where \(G(X)=(\mathcal A(X)-b,X)\) and \(\mathcal K=\{0^m\}\times \mathbb S_+^n\).

For \(X\in \mathbb S^n\), \(\mu = (y,Z)\in\mathcal Y=\mathbb R^m\times \mathbb S^n\), \[ \begin{array}{rcl} L(X,y,Z)&=&\frac{1}{2}||X-B||^2-\langle y,b-\mathcal A(X)\rangle -\langle Z,X\rangle\\ &=& \langle y,b\rangle +\frac{1}{2}||B||^2+\frac{1}{2}||X||^2-\langle \mathcal A^*(y)+Z+B,X\rangle, \end{array} \] Thus \(\min\{L(X,y,Z)\;|\; X\in\mathbb S^n\}\) can be found by setting, \[ \nabla _XL(X,y,Z)=X-(B+Z+\mathcal A^*(y))=0\implies X=(B+Z+\mathcal A^*(y)). \] Then we have \[ \begin{array}{rcl} \theta (y,Z)&=&\min\{L(X,y,Z)\;|\; X\in\mathbb S^n\}\\ &=& \langle y,b\rangle +\frac{1}{2}||B||^2-\frac{1}{2}||B+Z+\mathcal A^*(y)||^2. \end{array} \] The dual problem is given by \[ \text{(D)}\quad \begin{array}{rCl} \max & \langle y,b\rangle +\frac{1}{2}||B||^2-\frac{1}{2}||B+Z+\mathcal A^*(y)||^2\\ \text{subject to}& (y,Z)\in\mathcal K^*. \end{array} \] Then we have \[ \text{(D)}\quad \begin{array}{rCl} \min & -\langle y,b\rangle+\frac{1}{2}||B+Z+\mathcal A^*(y)||^2\\ \text{subject to}& y\in\mathbb R^n\\ & Z\in \mathbb S_+^n. \end{array} \] Since we have \[ G=\Pi_{\mathbb S_+^n} (G)-\Pi_{\mathbb S_+^n} (-G), \]

then \[ \min\left\{||B+Z+\mathcal A^*(y)||\;|\; Z\in\ \mathbb S_+^n\right\} \\=||B+\mathcal A^*(y)+\Pi_{\mathcal S_+^n}(-(B+\mathcal A^*(y)))|| \\=||\Pi_{\mathbb S_+^n}(B+\mathcal A^*(y))||. \] Therefore, \[ \text{(D)}\quad \begin{array}{rCl} \min & \phi(y)=-\langle y,b\rangle+\frac{1}{2}||\Pi_{\mathbb S_+^n}(B+\mathcal A^*(y))||^2\\ \text{subject to}& y\in\mathbb R^n. \end{array} \] In this case, the dual problem is unconstrained problem. Therefore, we can solve it by finding the root of the gradient, \[ 0=\nabla \phi(y)=-b+\mathcal A \Pi_{\mathbb S_+^n}(B+\mathcal A^* (y)). \] Example (Sparse regression problem) \[ \min\{\frac{1}{2}||Ax-b||^2+\lambda ||x||_1\;|\; x\in\mathbb R^n\}, \] where \(A\in\mathbb R^{m\times n}\) and \(b\in\mathbb R^m\) are given data. Let \(u=b-Ax\), then we have \[ \text{(P)}\quad \begin{array}{rCl} \min & f(u)+g(x)\\ \text{subject to}& u+Ax=b \end{array} \] where \(f(u)=\frac{1}{2}||u||^2\) and \(g(x)=\lambda ||x||_1\).

The Lagrangian function is given as \[ \begin{array}{rcl} L(u,x,\xi)&=&f(u)+g(x)+\langle \xi,b-u-Ax\rangle\\ &=& f(u)-\langle \xi,u\rangle +g(x)-\langle A^T\xi,x\rangle +\langle \xi,b\rangle. \end{array} \] The Lagrangian dual function is given as \[ \theta(\xi)=\langle \xi,b\rangle+\min_u(f(u)-\langle \xi,u\rangle)+\min_x(g(x)-\langle A^T\xi,x\rangle)\\ =\langle \xi,b\rangle-\max_u(\langle \xi,u\rangle-f(u))-\max_x(\langle A^T\xi,x\rangle-g(x)). \] Both of the last two terms are conjugate functions. Therefore we have \[ \theta(\xi)=\langle \xi,b\rangle-\frac{1}{2}||\xi||^2-\delta_{B_\lambda}(A^T\xi). \] Therefore, the dual problem is \[ -\min -\langle\xi,b\rangle +\frac{1}{2}||\xi||^2+\delta_{B_\lambda}(A^T\xi). \] Definition (Lagrange Multipliers) \[ L(x,\mu)=f(x)-\langle \mu, G(x)\rangle,\quad x\in\mathcal X,\mu\in\mathcal Y, \] where \(L\) be the Lagrangian function for (COP). Assume that \(f\) and \(G\) are continuously differentiable. We say that \(\bar \mu \in\mathcal Y\) is a Lagrange multiplier of (COP) at a feasible point \(\bar x\), if it satisfies the KKT condition, \[ 0=\nabla _xL(\bar x,\bar \mu)=\nabla f(\bar x)-\nabla G(\bar x)\bar \mu,\quad 0\in \bar \mu+\mathcal N_\mathcal K(G(\bar x)), \] where \(\mathcal N_\mathcal K (G(\bar x))\) is the normal cone of \(\mathcal K\) at \(G(\bar x)\in \mathcal Y\). We denote the set of all Lagrange multipliers at \(\bar x\) as \(\mathcal M(\bar x)\). We call the pair \((\bar x,\bar \mu)\) a KKT point.

Remark

  1. If \(\mathcal C\) is a closed convex cone, then \(\mathcal K = 0^{\mathcal U}\times \mathcal C\) is a closed convex cone. \[ \begin{array}{rcl} -\bar \mu \in\mathcal N_\mathcal K(G(\bar x))&\iff& G(\bar x)\in\mathcal K,&\langle -\bar \mu,d-G(\bar x)\rangle \le 0,&&\forall d\in\mathcal K \end{array} \] We Choose \(d=0\) and \(d=G(\bar x)\). Then we have \[ \begin{array}{rcl} -\bar \mu \in\mathcal N_\mathcal K(G(\bar x))&\iff& G(\bar x)\in\mathcal K,&\langle \bar \mu,G(\bar x)\rangle = 0,&\langle \bar \mu,d\rangle\ge 0,&\forall d\in\mathcal K\\ &\iff& G(\bar x)\in\mathcal K,&\langle \bar \mu,G(\bar x)\rangle = 0,&\bar \mu\in \mathcal K^*. \end{array} \] In this case, the KKT conditions can be written as \[ (G(\bar x)\in\mathcal K)\perp (\bar \mu \in\mathcal K^*). \]

  2. The set \(\mathcal M(\bar x)\) may be empty or unbounded.

Example

For basic NLP, let \(\bar \mu=(\lambda,\rho)\), and \(G(x)=(g(x),h(x))\), \(\mathcal C=\mathbb R^p_+\), and \(\mathcal K=\{0^m\}\times \mathcal C\), we have KKT conditions, \[ \nabla f(\bar x)-\nabla g(\bar x)\lambda -\nabla h(\bar x)\rho=0\\ G(\bar x)=(g(\bar x),h(\bar x))\in \mathcal K\\ \langle h(\bar x),\rho\rangle =0\\ \rho \in \mathcal C^*=\mathbb R_+^p. \] Example

Given \(B\in\mathbb S^n\), consider the doubly nonnegative projection problem, \[ \min \left\{\frac{1}{2}||X-B||^2\;|\; X\in\mathbb S_+^n\cap \mathcal N^n\right\}. \] The above problem can be rewritten as \[ \text{(D)}\quad \begin{array}{rCl} \min & f(X)=\frac{1}{2}||X-B||^2\\ \text{subject to}& G(X)=(X,X)\in\mathcal K=\mathbb S_+^n\times \mathcal N^n. \end{array} \] The Lagrange function is for \(\mu=(Y,Z) \in \mathcal Y=\mathbb S^n\times \mathbb S^n\), \[ \begin{array}{rcl} L(X,\mu)=L(X,Y,Z)&=&\frac{1}{2}||X-B||^2-\langle \mu,G(X)\rangle\\ &=&\frac{1}{2}||X-B||^2-\langle Y,X\rangle-\langle Z,X\rangle. \end{array} \] The Lagrange dual function is \[ \begin{array}{rcl} \theta(\mu)=\theta(Y,Z)&=&\min_{X\in\mathbb S^n} L(X,\mu)\\ &=&\min_{X\in\mathbb S^n}\left(\frac{1}{2}||X-B||^2-\langle Y,X\rangle-\langle Z,X\rangle\right)\\ &=&\min_{X\in\mathbb S^n}\left(\frac{1}{2}||X||^2-\langle B+Y+Z,X\rangle+\frac{1}{2}||B||^2\right)\\ &=&\frac{1}{2}\left(-||B+Y+Z||^2+||B||^2\right) \end{array} \]

Therefore the dual problem is \[ \text{(D)}\quad \begin{array}{rCl} \max & \frac{1}{2}\left(-||B+Y+Z||^2+||B||^2\right)\\ \text{subject to}& (Y,Z)\in\mathcal K^*=\mathbb S^n_+\times \mathcal N^n. \end{array} \] Therefore, the KKT conditions are given as \[ 0=\nabla _xL(\bar X,\bar Y,\bar Z)=\nabla f(\bar X)-\nabla G(\bar X)(\bar Y,\bar Z)\\ \mathcal K\ni G(\bar X)\perp (\bar Y,\bar Z) \in \mathcal K^*. \] That is (?????????) \[ \bar X- B-\bar Y-\bar Z=0\\ \langle \bar X,\bar Y\rangle=0\\ \langle \bar X,\bar Z\rangle =0 \\ \bar X\in \mathbb S^n_+\cap \mathcal N^n\\ \bar Y\in\mathbb S^n_+\\ \bar Z\in \mathcal N^n. \]

7.1 Robinson’s constraint qualification

In order for the KKT conditions to be necessary for the local minimizer \(\bar x\), we need to introduce some conditions to ensure the existance of Lagrange multipliers and the boundedness of \(\mathcal M(\bar x)\).

Definition (Robinson’s constraint qualification)

Let \(\bar x\) be a feasible solution to COP. Robinson’s constraint qualification is said to hold at \(\bar x\) if \[ 0\in\text{int}\left(G(\bar x)+G'(\bar x)\mathcal X-\mathcal K\right). \]
