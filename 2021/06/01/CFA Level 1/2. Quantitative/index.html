2. Quantitative methods

Reading 6 Time Value of Money

1. Decomposition of Required Rate of Return

Fisher Effect

  • \((1+R_n)=(1+R_r)(1+I)\)
    • \(R_n \approx R_r+I\)
    • \(R_n\): Nominal interest rate
    • \(R_r\): Real interest rate
    • \(I\): Expected inflation rate

Required Interest Rate on Security

  • \(R_i=R_f+R_p\)
    • \(R_i\): Required rate of return
    • \(R_f\): Nominal risk-free rate
    • \(R_p\): Risk premium
      • Default risk premium: 拖延或者还的少
      • Liquidity risk premium: 流动性差风险高
      • Maturity risk premium: 期限风险，时间长风险越大

2. Effective Annual Rate (EAR)

Calculation

  • \(EAR=\left(1+\frac{r}{m}\right)^m-1\)

Continuous Compounding

  • \(EAR = e^r-1\)

3. Annuities

Calculation

  • \(FV=PV\left(1+\frac{r}{m}\right)^{mn}\)

Requirements

  • Same intervals
  • Same amount of cash
  • Same direction

Classification

  • Ordinary annuities

    • 普通年金，后付年金

  • Annuity due

    • 先付年金

  • Perpetuity

    • 永续年金 \(PV=\frac{A}{r}\)

Reading 7 Statistical Concepts and Market Returns

1. Holding Period Return (HPR)

  • \(HPR=\frac{FV-PV}{PV}=\frac{P_1-P_0+CF}{P_0}\)

    • \(P_1-P_0\)：价差

    • \(CF\)：稳定现金流（比如分红，股利）

    • 不是年化收益率

2. Statistics Classification

  • Descriptive statistics
  • Inferential statistics
    • Population 总体
      • Parameter
    • Sample 样本
      • Sample statistic
      • Sample size 样本容量

3. Measurement Scales

  • Types of measurement scales:
    • Nominal scales 类别，男女
    • Ordinal scales 大小，排名
    • Interval scales 加减，温度
    • Ratio scales 乘除，金钱

4. Measure of Central Tendency

  • Arithmetic mean, geometric mean, harmonic mean

    • 算数平均>=几何平均>=调和平均

      拿123试一试

    • 调和平均：算平均估股价

      \[ \frac{3}{1/p_1+1/p_2+1/p_3} \]

  • 算数平均看的是未来，Expectation是一种算数平均

    • 几何平均衡量历史业绩

5. Quantiles

  • Quartile 四分位

  • Quintile 五分位

  • Deciles 十分位

  • Percentile 百分位

  • 位置：第 \((n+1)*分数\) 个数字

6. Measures of Dispersion

  • Mean absolute deviation (MAD)

    • \(MAD=\frac{\sum|X_i-\bar X|}{n}\)

  • Variance & standard deviation

    • For population \(\sigma\)
    • For sample \(s\)

7. Chebyshev’s Inequality

  • \(P(\mu-k\sigma\le X\le\mu+k\sigma)\ge 1-\frac{1}{k^2}\)

8. CV & SR

Coefficient of Variation (CV)

  • \(CV=\frac{s}{\bar X}\)

  • 每单位均值的标准差

  • Scale-free (没有单位)

  • Relative dispersion

  • Example 修路，没有考虑规模因素

Sharp Ratio (SP)

  • \(SR=\frac{R_p-R_f}{\sigma_p}\)

  • 每单位风险的超额收益，所以越大越好

9. Skewness

  • 定义

    • Positive (right) skew 尾巴在右边

  • Mode<median<mean

  • 投资者喜欢右偏：小概率获得大收益

  • 阶数 = 3

10. Kurtosis

  • Leptokurtic 高峰 Platykurtic 低峰

  • Normal distribution的kurtosis=3，大于3是高峰

  • 阶数=4

  • 高峰肥尾:

    假设：和正态分布离散程度相同

    结论：不喜欢高峰肥尾

Reading 8 Probability Concepts

1. Probability Classification and Odds

  • Objective probability
    • 通过客观数据
    • Empirical probability
      • 特定事情发生次数占总次数
    • Prior probability
      • 基于历史数据和逻辑分析
  • Subjective probability
    • Intuitive estimation
  • Odds for an event
    • 发生概率/不发生概率

2. Multiplication and Addition Rule

  • Multiplication

    • \(P(AB)=P(A|B)P(B)=P(B|A)P(A)\)

  • Addition

    • \(P(A\ or\ B) = P(A)+P(B)-P(AB)\)

  • Independent even

    • \(P(AB)=P(A)P(B)\)

  • Exclusive

    • \(P(AB)=0\)

3. Expected Value and Variance

  • Expected value

    • \(E(X)=\sum P_iX_i\)

  • Variance

    • \(\sigma^2 = E\Big[\big(X_i-E(X)\big)^2\Big]\)

4. Covariance and Correlation

  • Covariance
    • \(COV(X,Y)=E\Big[\big(X-E(X)\big)\big(Y-E(Y)\big)\Big]=\sigma_X\sigma_Y\rho_{XY}\)
  • Correlation
    • \(\rho_{XY}=\frac{COV(X,Y)}{\sqrt{Var(X)Var(Y)}}=\frac{COV(X,Y)}{\sigma_X\sigma_Y}\)
      • range：\([-1,1]\)

5. Portfolio Return and Risk

  • Return

    • \(E(r_p)=\sum_{i=1}^n w_iE(R_i)\)

  • Variance

    • Two assets Variance

      \(COV(X_1,X_2)=w_1\sigma_1^2+w_2\sigma_2^2+2w_1w_2\sigma_1\sigma_2\rho_{1,2}\)

6. Bayes’ Formula

  • Definition \[ P(AB)=P(A)P(B|A)=P(B)P(A|B)\\ P(A|B)=\frac{P(A)P(B|A)}{P(B)}\\ P(B)=P(A_1)P(B|A_1)+P(A_2)P(B|A_2)+\dotsm \]

7. Principle of Counting

  • Combination
    • Notation: \(nCr\)
  • Permutation
    • Notation: \(nPr\)

5. Safety First Ratio

Short Fall Risk Definition

  • Minimum return required 低于客户要求的最小回报率

Safety First Ratio Definition

  • S-F-Ratio\(=\frac{E(R_p)-R_L}{\sigma_p}\)
    • 当\(R_L = R_f\), 就是sharp ratio
    • 越大越好，S-F-Ratio越大，客户遇到short fall risk的可能性越小

6. Lognormal Distribution

Definition

  • 如果 \(\ln X\sim N\), \(X\sim \log N\)

Features

  • 右偏

  • 随机变量大于0

Application

  • 一般假设价格服从对数正态分布

  • 一般假设return服从正态分布

Reading 10 Sampling and Estimation

1. Sampling

Sampling Classification

  • Simple sampling 简单抽样
  • Stratified sampling 分层抽样

Sampling Rrror

  • Definition

    • Sample mean - population mean

  • 降低 sampling error 方法：增加样本容量

Types of Data

  • Time series data 同一公司不同时间
  • Cross-sectional data 同一时间不同公司

2. Central Limit Theory

Assumption

  • 样本容量 \(n\) 足够大 (\(n\ge 30\))
  • 总体的 \(\mu\) 和 \(\sigma^2\)已知

Conclusion

  • \(\bar X\sim N(\mu_x,\frac{\sigma^2}{n})\)

Standard Error

  • 样本均值的标准差 \(\sigma_{\bar X}=\frac{\sigma_x}{\sqrt n}\) 叫做 standard error （标准误）
    • 如果总体的variance未知，可以用样本的variance代替

3. Estimation

Properties

  • Unbiasedness
    • $E(X)=$
  • Efficiency
    • 方差小
  • Consistency
    • 当样本容量增加，准确性应该增加
    • 当样本容量增加，standard error应该下降

Estimation Classification

  • Point estimation

    • \(\mu=\bar x\)

  • Confidential interval (CI) estimation 置信区间估计

    • \(\bar X \pm k \frac{\sigma}{\sqrt n}\)

    • \(k\) 对应的百分比叫做 degree of confidence 置信度

    • 剩下的尾巴部分面积叫做level of significance 显著性水平，通常用 \(\alpha\) 表示

5. Student t Distribution

Features

  • Symmetric
  • 唯一的参数：Degrees of freedom (\(df\))
    • \(df=n-1\)
  • 相比于正态分布，低峰肥尾
  • 当 \(df\) 足够大，趋近于标准正态分布

Z-Distribution V.S. t-Distribution

  • 方差已知用z
  • 方差未知用t
  • 非正态小样本不可估计
  • 大样本 (\(n\ge 30\)) 任何情况可用z

6. Types of Bias

  • Data-mining bias
    • 偶然当必然
    • 如果用相同数据库测试和训练，会产生该偏差
  • Sample selection bias
    • 例子：survivorship bias 幸存者偏差
  • Look ahead bias
    • 用到的数据得不到，用历史数据估计产生的偏差
  • Time-period bias
    • 与时间相关的偏差

Reading 11 Hypothesis Testing

1. Critical Value Method

State Hypothesis

  • Null hypothesis (\(H_0\))
    • 想拒绝的假设 (等于号都放在这里边)
  • Alternative hypothesis (\(H_a\))

    • 想接受的假设

    • One tailed and two tailed

      • One tailed
      • \(H_0:\mu=\mu_0\quad H_a:\mu\neq \mu_0\)
      • Two tailed
        • \(H_0: \mu\le \mu_0\quad H_a: \mu>\mu_0\)
        • \(H_0: \mu\ge \mu_0\quad H_a: \mu<\mu_0\)

Test Statistic

  • Calculation

    • \(\frac{\bar X-\mu}{\sigma/\sqrt n}\)

  • Critic Value Selection (就是那个k)

    • 与 \(\alpha\) 有关

    • 与查哪张表有关

    • 与单尾双尾有关

  • Statement

    • Reject the null hypothesis

    • Fail to reject the null hypothesis

2. Hypothesis Testing for Different Parameters

Mean Hypothesis Testing

  • 一个均值
    • z/t
  • 两个均值之间的关系 t分布
    • 独立的
    • 不独立
      • paired comparison test
      • 假设检验差值是否等于0

Variance Hypothesis Testing

  • 一个方差：\(\Chi^2\) 分布
  • 两个方差：F 分布

Correlation Hypothesis Testing

  • Null hypothesis \(H_0: r=0\)

  • \[ t=\frac{r-0}{\sqrt{\frac{1-r^2}{n-2}}}\\ df=n-2 \]

  • r 是相关系数

  • 如果拒绝了原假设，也就是说 \(r\) significantly different from 0，说明有线性关系

3. P-Value Method

Definition

  • 拒绝原假设的最小显著性水平

Compare \(P\) & \(\alpha\)

  • \(P>\alpha\) 不能拒绝原假设

  • P越小越拒绝

4. Type I and Type II error

Definition

  • 原假设正确但是reject了，第一类错误，错杀好人

  • \(P(\text{I error})=\alpha\)

  • 原假设错误但是fail to reject了，第二类错误，放走坏人

Power of test

  • 原假设是错误的，拒绝了

  • \(1-P(\text{II error})\)

Features

  • 当n不变，I error 和 II error此消彼长
  • 当n增大，I和II error都下降

5. Nonparametric test

  • 对非参数检验
    • 当数据不满足分布假设
    • 当数据是排序的
    • 当检验的不是参数
