3. Fundamental of Lyapunov Theory

  1. There are two methods to solve the stability of a system proposed by Lyapunov. Linearization method and direct method.

  2. Nonlinear system can be represented by \[ \dot{\mathbf x} = \mathbf f(\mathbf x,t) \] and if there is input \(u\) \[ \dot{\mathbf x}= \mathbf f(\mathbf x,\mathbf u,t) \] and if \(u\) depends on \(x\) and \(t\) ,we have \[ \dot{\mathbf x}= \mathbf f(\mathbf x,\mathbf g(\mathbf x,t),t) \] if there is no control signal, the first differential equation is used.

  3. Autonomous and non-autonomous system

    The system is said to be autonomous if \(\mathbf f\) does not depend explicitly on time \(t\) \[ \dot{\mathbf x} = \mathbf{f}(\mathbf x) \] otherwise, the system is called non-autonomous.

    obviously, LTI systems are autonomous and LTV are non-autonomous system

    autonomous systems and non-autonomous systems, time-varying systems and time invariant systems are the definition on the closed-loop dynamics. Therefore, they can be due to the plant of the control law.

    The fundamental difference between autonomous system and non-autonomous systems is the former is independent of the initial time.

  4. Equilibrium points

    Equilibrium points are the state of the system if once \(x(t)\) is equal to \(x^*\) , it remains to \(x^*\) for all future time.

    Because of \[ \tag{3.1} \dot{\mathbf x} = \mathbf f(\mathbf x) \] We have \[ \mathbf f(\mathbf x^*)= \mathbf 0 \] For a LTI system \[ \dot{\mathbf x}=A\mathbf x\\ A\mathbf x^*=0 \] if \(A\) is nonsingular, that means there is no nontrivial solution for the equation, there is only trivial solution \(\mathbf x^*=0\) , there is only one equilibrium for the system and if \(A\) is singular, there are equilibriums same as the solutions for the above equation.​

​ and if the equilibrium is not at the origin, we can change transfer it to the origin. \[ \mathbf y = \mathbf x - \mathbf x^*\\ \mathbf x= \mathbf y + \mathbf x^* \\ \dot{(\mathbf y + \mathbf x^*）}=\mathbf f(\mathbf y + \mathbf x^*) \\ \dot{\mathbf y}=\mathbf f(\mathbf y + \mathbf x^*) \]

  2. Concepts of stability

    Definition 3.3 \[ \forall R>0, \exist r>0,||x(0)||<r\implies \forall t \ge0, ||x(t)||<R \] The system is stable.

    Inverse \[ \forall R>0, \exist r>0,||x(0)||<r\implies \forall t \ge0, ||x(t)|| \ge R \] The system is unstable.

    Asymptotic stability, the system is stable and then \[ \exist r>0, ||x(0)||<r \implies x \rightarrow \infty ,x(t)\rightarrow 0 \] State converge does not imply stability

    exponential stability \[ \forall t>0, \exist \alpha\space\lambda,||x(t)||\le\alpha||x(0)||e^{-\lambda t} \] exponential stability implies asymptotic stability

    local and global stability

    if the asymptotic stability holds for any initial states, it is called global stability

  3. Linearization and local stability \[ \dot{x}=(\frac{\partial f}{\partial x})_{x=0}x+O(f(x)) \]

    \[ A = (\frac{\partial f}{\partial x})_{x=0}x \]

    \[ \tag{3.2} \dot{x}=Ax \]

    that is called the linearization (linear approximation) of the original nonlinear system.

    and of \[ \dot{x}=f(x,u) \\ \dot{x}=(\frac{\partial f}{\partial x})_{x=0,u=0}x+(\frac{\partial f}{\partial u})_{x=0,u=0}u+O(f(x,u))\\ A = (\frac{\partial f}{\partial x})_{x=0,u=0}x \\ B = (\frac{\partial f}{\partial u})_{x=0,u=0}u \] and if the control law of the system is \(u=u(x)\), which is autonomous, and\(u(0)=0\) \[ u=u(x)=\frac{\partial u}{\partial x}_{x=0}u=Gx \\ G = \frac{\partial u}{\partial x}_{x=0}u\\ \dot{x}=f(x,u)=(A+BG)x \] Lyapunov's Linearization method

    if the linearized system is strictly stable, then the equilibrium point is asymptotic stable

    if the linearized system is strictly unstable, then the equilibrium point is unstable

    if the linearized system is marginally stable, one cannot make conclusion.

  4. Lyapunov's Direct Method

    if the total energy of the system is continuously dissipated, then the system will eventually settle down to an equilibrium point.

    some conclusions

  5. Positive definite functions and Lyapunov Functions

    The energy functions have two properties. 1> it is strictly positive unless all state variables are zero. 2> the function is monotonically decreasing when the states varying according to the state functions. The first property is the positive definite functions. The second one is called Lyapunov Functions.

    Locally Positive Definite, if \(V(0)=0\) and in a ball \(B_{R_{0}}\) \[ x \neq 0 \implies V(x)>0 \] Globally Positive Definite, if the above property holds for the whole state space.

    \(V(x)\) is negative definite if \(-V(x)\) is positive definite

    \(V(x)\) is positive semi-definite if $V(x) 0 $ for \(x \neq 0\)

    \(V(x)\) is negative semi-definite if \(-V(x)\) is positive semi-definite. \[ \dot{V}=\frac{dV(x)}{dt}=\frac{\partial V}{\partial x}\frac{dx}{dt}=\frac{\partial V}{\partial x}f(x) \] if in a ball \(B_{R_{0}}\) , the function \(V(x)\) is positive definite and has continuous partial derivatives, and if derivative along any state trajectory of system is negative semi-definite \[ \dot{V(x)} \le 0 \] Then \(V(x)\) is said to be Lyapunov Function for the system.

  6. Equilibrium point theorems

    Lyapunov Theorem for local stability

    if, in a ball \(B_{R_{0}}\) ,there exists a scalar function V(x) with continuous first partial derivatives such that

    \(V(x)\) is positive definite

    \(\dot{V}(x)\) is negative semi-definite

    then the equilibrium is stable.

    \(\dot{V}(x)\) is negative definite

    then the equilibrium is asymptotic stable.

    Lyapunov Theorem for global stability

    \(\dot{V}(x)V(x)\) is positive definite

    \(\dot{V}(x)\) is negative definite

    \(V(x) \rightarrow \infty\) as \(||x|| \rightarrow \infty\)

    The last one is to ensure the closed contour of \(V(x)\)

    Many Lyapunov Functions exist for the same system. e.g. \[ V_1=\rho V^\alpha \] \(\rho\) is strictly positive definite and \(\alpha\) is a scalar.

  7. Invariant set theorems

    A set is an invariant set for a dynamic system if every system trajectory which starts from a point in G remains in G for all future time.

    Local invariant set theorem

    the invariant set theorem reflect the intuition that the decrease of a Lyapunov function V has to gradually vanish.

    assume

    for some \(l>0\) , the region \(\Omega _l\) defined by \(V(x)<l\) is bounded

    \(\dot{V}(x)\le 0\) for all \(x\) in \(\Omega_l\)

    Let R be the set of all point in \(\Omega_l\) where \(\dot{V}(x)= 0\) , and \(M\) be the largest invariant set in \(R\) . Then every solution \(x(t)\) originating in \(\Omega_l\) tends to \(M\) as \(t \rightarrow \infty\) .

    (if $(x)=0 $ for all future \(t\), then \(M=R\) )

    that means, if M only contains the origin, the system is locally asymptotic stable.

    \(V(x)\) is locally positive definite

    \(\dot{V}\) is negative semi-definite

    the set \(R\) defined by \(\dot{V}(x)=0\) only contains the origin.

    then the equilibrium is asymptotically stable.

    furthermore, the largest connected region of the \(\Omega_l\) (defined by \(V(x)<l\)) is a domain of attraction.

​ Globally invariant set theorem

​ \(\dot{V}(x) \le 0\) over the whole state space

​ \(V(x) \rightarrow \infty\) as \(||x|| \rightarrow \infty\)

​ Let R be the set of all points where \(\dot{V}(x) = 0\) , and M be the largest invariant set in R. Then all solutions globally asymptotically converge to M as \(t\rightarrow \infty\) .

  3. Krasovskii's Method

    Let \(A(x)\) denote the Jacobian matrix of the system. \[ A(x)=\frac{\partial f}{\partial x} \] if the matrix \[ F=A+A^T \] is negative definite in a neighborhood \(\Omega\)

    then the equilibrium point at the origin is asymptotically stable. A Lyapunov Function for the system is \[ V(x)=f^T(x)f(x) \] if \(\Omega\) is entire state space and in addition, \(V(x) \rightarrow \infty\) as $||x|| $, then the equilibrium point is globally asymptotically stable.

    proof:

    first, let us prove that the negative definiteness of F means \(f(x) \neq 0\) for \(||x|| \neq 0\)

    if \(A\) is singular, there is a \(y_0\) , we have \(Ay_0=0\) .

    then \[ y_0^TFy_0=y_0^TAy_0+y_0^TAy_0=2y_0^TAy_0=0 \] that means \(F\) is not negative definite, which contradicts the assume.

    then \[ \begin{array}{lll} \dot{V}(x)&=&\dot{f^T}(x)f(x)+f^T(x)\dot{f}(x) \\ &=& f^TA^Tf+f^TAf \\ &=& f^T(A^T+A)f \\ &=& f^TFf \end{array} \] That means if F is negative definite, \(\dot{V}(x)\) is negative definite.

  6. Lyapunov-Like Analysis using Barbalat's Lemma

    \(\dot{f} \rightarrow 0 \quad t \rightarrow \infty\) does not mean f converge

    f converge does not mean \(\dot{f} \rightarrow 0\)

    if \(f\) is lower bounded and decreasing (\(\dot{f} \le 0\)), then it converges to a limit.

    Barbalat's Lemma

    if differential function \(f(t)\) has a finite limit as $t $, and if \(\dot f(t)\) is uniformly continuous, then \(f(t) \rightarrow 0 \quad as \quad t \rightarrow \infty\)

    it can be proved that if $ (t)$ is bounded, then \(\dot f(t)\) is uniformly continuous.

    Lyapunov-Like Lemma

    if a scalar function \(V(x,t)\) satisfies the following conditions

    \(V(x,t)\) is lower bounded

    \(\dot V(x,t)\) is negative semi-definite

    \(\dot V(x,t)\) is uniformly continuous in time

    then \(\dot{V}(x,t) \rightarrow0\) as \(t \rightarrow \infty\)

    example

    for an adaptive control system \[ \dot e = -e+\theta w(t) \\ \dot \theta = -ew(t) \] Consider \[ V = e^2+\theta^2 \] We have \[ \begin{array}{} \dot V &=& 2e\dot e+2\theta \dot \theta \\ &=&2e(-e+\theta w(t))+2\theta(-ew(t)) \\ &=&-2e^2 \end{array} \]

    that means \(V(t) \le V(0)\) , \(e\) and \(\theta\) is bounded.

​ \(V\) is lower bounded and \(\dot V\) is negative semi-definite. we need \(\dot V\) is uniformly continuous. \[ \ddot V = -4e \dot e=-4e(-e+\theta w(t)) \]

​ \(\ddot V\) is bounded

​ \(\dot V\) goes to 0 as \(t \rightarrow \infty\)

​ that means $e 0 $ as \(t \rightarrow \infty\)

  1. Positive Linear System

    Positive Real Function and Strictly Positive Real Function

    a transfer function is positive real if \(Re[h(p)] \ge 0\) for all \(Re[p] \ge 0\)

    it is strictly positive real if \(h(p-\epsilon )\) is positive real for some \(\epsilon \ge 0\)

    A transfer function is SPR if and only if

    ​ 1) \(h(p)\) is a strictly stable transfer function

    ​ 2) the real part of \(h(p)\) is strictly positive along the \(j\omega\) axis, \(\forall \omega \ge 0\), \(Re[h(j \omega)]>0\)

    there are four necessary condition according to the above

    ​ 1) \(h(p)\) is strictly stable

    ​ 2) Nyquist plot of \(h(j\omega)\) lies entirely in the right half complex plane.

    ​ 3) \(h(p)\) has relative degree 0 or 1

    ​ 4) \(h(p)\) is strictly minimum-phase (all zeros are strictly in the left half plane)

  2. KY Lemma

    consider a controllable LTI system \[ \dot{x}=Ax+bu\\ y=c^Tx \] the transfer function \[ h(p)=c^T[pI-A]^{-1}b \] is SPR if and only if there exist positive definite matrices P and Q such that \[ A^TP+PA=-Q \\ Pb=c \]
