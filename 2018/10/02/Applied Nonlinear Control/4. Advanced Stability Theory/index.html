1. concepts of stability for non-autonomous systems

For non-autonomous system \[ \dot{x} = f(x,t) \] equilibrium points are defined as \[ f(x^*,t)=0 \] for \(t \in [t_0,\infty]\) .

Example \[ \dot x=-\frac{a(t)x}{1+x^2} \] Then the equilibrium point is \(x=0\).

==Stability== \[ \forall R>0 \quad \exist r(t_0)>0 \quad||x(t_0)||<r \implies ||x(t)||< R \quad \forall t\ge t_0 \] ==Asymptotically Stable==

it is stable and \[ \exist r(t_0) > 0 \] such that $$ \begin{array}{} ||x(t_0)||<r(t_0) & & ||x(t)|| 0 & as & t

\end{array} \[ ==Exponentially Stable== \] \begin{array}{} ||x(t)|| ||x_0||e^{-(t-t_0)} & t t_0

\end{array} $$

==Globally Asymptotically Stable== \[ x(t) \rightarrow0 \space as \space t\rightarrow \infty \] ==Uniformity== in stability concepts

the equilibrium is uniformly stable if the scalar \(r\) can be chosen independently of \(t_0\) \[ r=r(R) \] ==Uniformly Asymptotically Stable== \[ r=r(R) \]

Non-uniformity means less and less stable and longer and longer converge time with large \(t_0\) .

exponentially stable usually means uniformly asymptotically stable.

2. Lyapunov Analysis of Non-Autonomous system

==Locally Positive Definite== \[ \forall t \ge t_0 , \quad V(x,t) \ge V_0(x) \] \(V_0(x)\) is positive definite.

That means a time-variant function is locally positive definite if it dominates a time-invariant locally positive definite function.

Other concepts are the same.

==Decrescent==

A scalar function \(V(x,t)\) is decrescent if \(V(0,t)=0\) and if there exists a time-invariant positive definite function \(V_1(x)\) \[ \forall t \ge0, \quad V(x,t) \le V_1(x) \] for the time variant scalar function \(V\) \[ \frac{dV}{dt}=\frac{\part V}{\part t}+\frac{\part V}{\part x}\frac{\part x}{\part t}\\ =\frac{\part V}{\part t}+\frac{\part V}{\part x}f(x,t) \] Example

Consider the positive definite function \[ V(x,t)=(1+\sin^2t)(x_1^2+x_2^2) \] It dominates the function \(V_0=x_1^2+x_2^2\) and it is dominated by \(V_1=2(x_1^2+x_2^2)\) . Therefore the function is positive definite and is decrescent.

==Lyapunov Theorem for Non-Autonomous Systems==

for \(V(x,t)\) , in a ball \(B_{R_{0}}\)

  1. V is positive definite
  2. \(\dot{V}\) is negative semi-definite

then the equilibrium point 0 is stable.

If V is decrescent, uniformly stable.

If \(\dot{V}\) is negative definite and V is decrescent, uniformly asymptotically stable

If \(B_{R_{0}}\) is the whole state, globally uniformly asymptotically stable.

3. Proof of Lyapunov Analysis of Non-Autonomous Systems

A continuous function \(\alpha\) is said to be of class K (\(R^+ \rightarrow R^+\)) if

  • \(\alpha(0)=0\)
  • \(\alpha(p)>0\) if \(\forall p>0\)
  • \(\alpha\) is non-decreasing

Lemma 1

A function is locally (globally) positive definite if, and only if, there exists a function \(\alpha\) of class K such that \(V(0,t)=0\) and \[ V(x,t) \ge \alpha(||x||) \] A function is locally (globally) decrescent if and only if there exists a class K function \(\beta\) such that \(V(0,t)=0\) and
\[ V(x,t) \le \beta(||x||) \] Theorem

For non-autonomous system \(V(x,t)\) and a class K function \(\alpha\), \(\beta\), \(\gamma\)

  1. \(V(x,t) \ge \alpha(||x||) > 0\)

  2. \(\dot V(x,t) \le 0\)

The system is \(L\) stable.

  1. \(V(x,t) \le \beta(||x||)\)

The system is uniformly stable.

if 2 is change to

\(\dot V(x,t) \le -\gamma(||x||) < 0\)

the system is uniformly asymptotically stale

\(\displaystyle \lim _{x \rightarrow \infty} \alpha(||x||) \rightarrow \infty\) and 1. 2. 3. are satisfied in whole state space then

the system is globally uniformly asymptotically stable.

Consider the system defined by \[ \dot x_1=-x_1-e^{-2t}x_2\\ \dot x_2=x_1-x_2 \] Now we try \[ V=x_1^2+(1+e^{-2t})x_2^2\\ \dot V=-2e^{-2t}x_2^2+2x_1(-x_1-e^{-2t}x_2)+(1+e^{-2t})2x_2(x_1-x_2)\\ =-2[x_1^2-x_1x_2+(2e^{-2t}+1)x_2^2] \] Then we have \[ \dot V \le-2[x_1^2-x_1x_2+x_2^2]<0 \] System is globally asymptotically stable.

but this theorem is only sufficient

if origin is the equilibrium point \[ A(t)=(\frac{\part f}{\part x})_{x=0}\\ \dot{x}=A(t)x \] the second part is said to be the linearization of the non-autonomous system.

but \[ \dot{x}=A(t)x+O(x,t) \] the system can be linearized only with this condition satisfied \[ \lim_{||x||\rightarrow 0} \sup\frac{||O(x,t)||}{||x||}=0, \quad \forall \ge0 \] Note, \(A\) can be time-varying and also can be constant.

6. Instability Theorem

First we can try the linearization method to analyze the instability of the system.

when the linearization method failed. we can use the direct method.

==First Instability Theorem==

\(V(0,t) = 0 \quad \forall \space t\ge t_0\)

\(V(x,t_0)\) can assume strictly positive value arbitrarily close to the origin

\(\dot{V}(x,t)\) is positive definite.

then the system is unstable.

Example

Consider the system \[ \dot x_1=2x_2+x_1(x_1^2+2x_2^4)\\ \dot x_2=-2x_1+x_2(x_1^2+x_2^4) \] We take \[ V=\frac{1}{2}(x_1^2+x_2^2) \] its derivative is \[ \dot V=(x_1^2+x_2^2)(x_1^2+x_2^4) \] The system is instable.

==Second Instability Theorem== \[ \dot{V}(x,t)-\lambda V(x,t) \ge 0 \quad \forall t \ge t_0 \quad \forall x \in \Omega \] \(\Omega\) is a certain neighborhood of the origin. V is differentiable, decrescent function. \[ V(x,t)=x_1^2-x_2^2 \] For the above equation, \(\Omega\) can be said to be a certain neighborhood of the origin.

Stability

If the origin of the system is stable, there exists a positive definite function \(V(x,t)\) with a non-positive definite.

8. Lyapunov-Like Analysis Using Barbalat's Lemma

\(\dot{f} \rightarrow 0 \quad t \rightarrow \infty\) does not mean f converge

Example \[ f(t)=\sin (\log t)\\ f'(t)=\frac{\cos(\log t)}{t} \rightarrow0 \] f converge does not mean \(\dot{f} \rightarrow 0\)

\[ f(t)=e^{-t}\sin(e^{2t})\rightarrow0\\ f'(t)=-e^{-t}\sin (e^{2t})+e^{t}\cos (e^{2t}) \rightarrow\infty \] If \(f\) is lower bounded and decreasing (\(\dot{f} \le 0\)), then it converges to a limit.

==Barbalat's Lemma==

If differential function \(f(t)\) has a finite limit as $t $, and if \(\dot f(t)\) is uniformly continuous, then \(\dot f(t) \rightarrow 0\) as \(t \rightarrow \infty\)

It can be proved that if $ (t)$ is bounded, then \(\dot f(t)\) is uniformly continuous.

==Lyapunov-Like Lemma==

If a scalar function \(V(x,t)\) satisfies the following conditions

\(V(x,t)\) is lower bounded

\(\dot V(x,t)\) is negative semi-definite

\(\dot V(x,t)\) is uniformly continuous in time

then \(\dot{V}(x,t) \rightarrow0\) as \(t \rightarrow \infty\)

Example

for an adaptive control system \[ \dot e = -e+\theta w(t) \\ \dot \theta = -ew(t) \] Consider \[ V = e^2+\theta^2 \] We have \[ \begin{array}{} \dot V &=& 2e\dot e+2\theta \dot \theta \\ &=&2e(-e+\theta w(t))+2\theta(-ew(t)) \\ &=&-2e^2 \end{array} \]

that means \(V(t) \le V(0)\) , \(e\) and \(\theta\) is bounded.

\(V\) is lower bounded and \(\dot V\) is negative semi-definite. we need \(\dot V\) is uniformly continuous. \[ \ddot V = -4e \dot e=-4e(-e+\theta w(t)) \]

\(\ddot V\) is bounded

\(\dot V\) goes to 0 as \(t \rightarrow \infty\)

that means $e 0 $ as \(t \rightarrow \infty\)

9. Positive Linear System

In the analysis and design of nonlinear system, it is possible to decompose the system into a linear subsystem and a nonlinear subsystem. If the transfer function of the linear subsystem is so-called positive real, then it has important properties which may lead to the generation of a Lyapunov function for the whole system.

==Positive Real Function and Strictly Positive Real Function==

A transfer function is positive real if \(Re[h(p)] \ge 0\) for all \(Re[p] \ge 0\)

It is strictly positive real if \(h(p-\epsilon )\) is positive real for some \(\epsilon \ge 0\)

It means that h(p) always has a positive real part when p has positive (or zero) real part.

A transfer function is SPR if and only if

​ 1) \(h(p)\) is a strictly stable transfer function

​ 2) the real part of \(h(p)\) is strictly positive along the \(j\omega\) axis, \(\forall \omega \ge 0\), \(Re[h(j \omega)]>0\)

There are four necessary conditions according to the above

​ 1) \(h(p)\) is strictly stable

​ 2) Nyquist plot of \(h(j\omega)\) lies entirely in the right half complex plane.

​ 3) \(h(p)\) has relative degree 0 or 1

​ 4) \(h(p)\) is strictly minimum-phase (all zeros are strictly in the left half plane)

==KY Lemma==

Consider a controllable LTI system \[ \dot{x}=Ax+bu\\ y=c^Tx \] The transfer function \[ h(p)=c^T[pI-A]^{-1}b \] is SPR if and only if there exist positive definite matrices P and Q such that \[ A^TP+PA=-Q \\ Pb=c \]
