2. Computer Control and Discrete-Time Systems

Posted on 2018-11-14 In EE5104 Computer Control

1. Pole Cancellation

If the whole system is stable, then the poles and zeros can be cancelled out each other, then the reduced transfer function is good approximation of the original one.

If the common factors are related to the same variables, they must cancel out.

If the common factors are related to different variables, they should remain.

The best tools to check whether they can be cancelled out or not is time domain. Proof is omitted.

Example

---u------G1------v------G2------y \[ G_1=\frac{s-1}{s^2+s+1}\\ G_2=\frac{1}{s-1}\\ \] Then \[ G_1G_2=\frac{s-1}{(s^2+s+1)(s-1)}\\ \] ---u------G2------v------G1------y \[ G_2G_1=\frac{1}{s^2+s+1}\\ \]

2. Zero-order Hold Discrete instead of First Order Hold

Because you will never know what will happen in the next time. So we can only choose zero-order hold for the computer control system.

3. How to Solve the Differential Equation

\[ x(t)=\underbrace{e^{At}x(0)}_{\text{zero input response}}+\underbrace{\displaystyle \int _0^te^{A(t-\tau)}Bu(\tau)d\tau}_{\text{zero-state response}}\\ \]

For continuous time, we have differential equation, differentiation operator, Laplace Transform, continuous state space.

For discrete time, we have difference equation, shift operator, Z Transform, discrete state space.

4. Discrete with Zero-Order Hold

For a constant in the whole interval \([t_k,t_{k+1})\) \[ x(t)=e^{A(t-t_k)}x(t_k)+\int_{t_k}^{t}e^{A(t-\tau)}d\tau Bu(t_k)\\ \] For the second part we have \(v=t-\tau\) \[ \int_{t_k}^{t}e^{A(t-\tau)}d\tau=\int_0^{t-t_k}e^{Av}dv\\ \] If the sampling time is h, we have \[ x(kh+h)=e^{Ah}x(kh)+\int_0^he^{Av}dvBu(kh)=\Phi x(kh)+\Gamma u(kh)\\ \] To be simple \[ x(k+1)=\Phi x(k)+\Gamma u(k)\\ y=Cx\\ \] where \[ \Phi=e^{Ah} \\ \Gamma=\int_0^he^{Av}dvB\\ \]

Difference Equation \[ y(k+1)=a_1y(k)+a_2y(k-1)+b_0u(k)+b_1u(k-1)\\ y(k+1)-a_1y(k)-a_2y(k-1)=b_0u(k)+b_1u(k-1)\\ \] We define shift-operator q \[ qf(k)=f(k+1)\\ q^{-1}f(k)=f(k-1)\\ \] Then we will have \[ qy(k)-a_1y(k)-a_2q^{-1}y(k)=b_0u(k)+b_1q^{-1}u(k)\\ \frac{y(k)}{u(k)}=\frac{b_0+b_1q^{-1}}{q-a_1-a_2q^{-1}}=\frac{b_0q+b_1}{q^2-a_1q-a_2}=\frac{B(q)}{A(q)}\\ \]

5. Z Transform

In continuous time domain, we have \[ u=e^{st}\quad y=H(s)e^{st}\\ \] In discrete time domain, we have \[ e^{st\quad\underrightarrow{t\rightarrow kh}}\quad e^{skh\quad\underrightarrow{e^{sh}\rightarrow a}} \quad a^k\\ \] Therefore, \[ u(k)=a^k\quad y(k)=H(a)a^k\\ \] Then we have the following calculation which has the same properties. \[ F(z)=\sum^\infty_{k=0}f(k)z^{-k}\\ \] Which is Z transform.

For impulse signal, we have \(U(z)=1\).

For \(u=a^k\), we have \(U(z)=\frac{1}{z-a}\).

Inverse Z Transform e.g. \[ z^2Y(z)-Y(z)=zU(z)+2U \\ \frac{Y}{U}=G(z)=\frac{z+2}{z^2-1}\\ \frac{G(z)}{z}=\frac{z+2}{z(z+1)(z-1)}=\frac{A_1}{z}+\frac{A_2}{z+1}+\frac{A_3}{z-1} \\ G(Z)=\frac{z+2}{z(z+1)(z-1)}=-1+\frac{0.5z}{z+1}+\frac{0.5z}{z-1} \\ \] After inverse z transform \[ y(k)=-\delta(k)+0.5(-1)^k+0.5 (1)^k \] Apply z transform to state space equation \[ x(k+1)=\Phi x(k)+\Gamma u(k)\\ y(k) = cx(k) \\ zX(z)-x(0)=\Phi X(z)+\Gamma U(z) \\ (zI-\Phi)X(z)=x(0)+\Gamma U(z) \\ X(z)=(zI-\Phi)^{-1}x(0)+(zI-\Phi)^{-1}\Gamma U(z) \\ Y(z)=cX(Z)=c(zI-\Phi)^{-1}x(0)+c(zI-\Phi)^{-1}\Gamma U(z)\\ \]

For s domain, steady state gain(SSG) is H(0).

For z domain, SSG is H(1).

Poles in continuous time domain(CTD) is \(\lambda\) , in the discrete time is \(e^{h\lambda }\)

In CTD, if \(\lambda = \sigma+j\omega\) , \(e^{\sigma+j\omega}=e^{\sigma}(cos (\omega)+jsin(\omega))\)

In DTD, \(e^{h\lambda}=e^{h(\sigma+jw)}=e^{h\sigma}e^{jhw}\)

Therefore, we can map poles from continuous time domain to discrete time domain.

1. Laplace Transform

Here we need to talk about Laplace Transform firstly. For the system \[ \ddot y-y=u \] if \(u=e^{st} \quad y=H(s)e^{st}\)

we can build the relationship between u and y. \[ s^2H(s)e^{st}-H(s)e^{st}=e^{st} \] Then we can get the following equation \[ H(s)=\frac{1}{s^2-1} \] The meaning of this equation is quite clear. If we choose one input \[ u=e^{2t} \] Obviously, s=2.

The output will be \[ y=H(2)e^{2t}=\frac{1}{3}e^{2t} \] Now we will choose another general tool which has the same properties. \[ u(t)=\frac{1}{2\pi i}\int_{a-i\infty}^{a+i\infty}U(s)e^{st}ds \] And then we can get U(s) \[ U(s)=\int_0^\infty u(t)e^{-st}dt \] This is the famous Laplace Transform. Fourier Transform transforms domain form time to frequencies. Laplace transform is another kind of transformation which may not have very clear physical meaning. But it has the very convenient calculate properties.

2. Stability

Stable: small deviation from the equilibrium will remain small.

Asymptotic Stable: small deviation from the equilibrium point will be completely eliminated.

Unstable: small deviation from the equilibrium point will result in huge deviation.

We will introduce a invert pendulum to this introduction part.

For a invert pendulum, we have the following equations derived from Newton's Second Law. \[ ml^2\ddot \theta=u(t)-mlgsin(\theta(t)) \] To make the problem simple, we have \[ ml^2=mgl=1\\ \ddot \theta+sin(\theta(t))=u(t) \] We can find two equilibrium points from physics \[ \theta=0,\dot \theta=0\quad\theta=\pi,\dot \theta=0 \] What is equilibrium in math?

Equilibrium points are the state of the system if once \(x(t)\) is equal to \(x^*\) , it remains to \(x^*\) for all future time.

Because of \[ \tag{3.1} \dot{\mathbf x} = \mathbf f(\mathbf x) \] We have \[ \mathbf f(\mathbf x^*)= \mathbf 0 \] For a LTI system \[ \dot{\mathbf x}=A\mathbf x\\ A\mathbf x^*=0 \] If \(A\) is nonsingular, that means there is no nontrivial solution for the equation, there is only trivial solution \(\mathbf x^*=0\) , there is only one equilibrium for the system and if \(A\) is singular, there are equilibriums same as the solutions for the above equation.​

And if the equilibrium is not at the origin, we can transfer it to the origin. \[ \begin{array}{lll} \mathbf y = \mathbf x - \mathbf x^*\\ \mathbf x= \mathbf y + \mathbf x^* \\ {(\mathbf y + \mathbf x^*）'}=\mathbf f(\mathbf y + \mathbf x^*) \\ \dot{\mathbf y}=\mathbf f(\mathbf y + \mathbf x^*) \end{array} \] Then we can say, for the invert pendulum, we have the following equations to get the equilibrium points. \[ \ddot \theta=0\quad\dot\theta=0\quad u=0 \] How to check whether the point is stable or not. Many methods. Here we talk about linearization.

With the help of Taylor Series, we can easily find linear equations for a nonlinear system in one point and its close area. Therefore, we will use Taylor Series to expand the invert pendulum system. Then we can get the following form. \[ \ddot y(t)-y=u(t) \]
