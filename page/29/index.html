Chapter 9. Clustering and Applications

Posted on 2019-04-26 In EE5907 Pattern Recognition

Clustering and Applications

1. What is Cluster Analysis

Finding groups of objects such that the objects in a group will be similar to one another and different from the objects in other groups.

A clustering is a set of clusters.

Partitional Clustering: A division of data objects into non-overlapping subsets, such that each data object is in exactly one subset.

Hierarchical clustering: A set of nested clusters organized as a hierarchical tree.

Exclusive and non-exclusive: In non-exclusive clustering, points may belong to multiple clusters.

Fuzzy and non-fuzzy: In fuzzy clustering, a point belongs to every cluster with some weight between 0 and 1.

Partial and Complete: In some cases, we only want to cluster some of the data.

2. K-means

  1. partitional clustering approach
  2. Each cluster is associated with a centroid
  3. Each point is assigned to a cluster with the closest centroid
  4. Number of clusters, K, must be specified

Algorithm

Select K points as the initial centroids

repeat

From K clusters by assigning all points to the closest centroid.

Recompute the centroid of each cluster.

until the centroid don't change

3. Evaluating K-means Clusters

Most common measure is Sum of Squared Error (SSE) \[ SSE=\sum_{i=1}^K \sum_{x\in C_i}dist^2(m_i,x) \] the initial chosen points are very important.

Solutions to Initial centroids problem

  1. Multiple runs
  2. Use hierarchical clustering to determine initial centroids
  3. Select more than K initial centroids and then select among these initial centroids (Select most widely separated)
  4. Postprocessing
    1. Eliminate small clusters that may represent outliers
    2. Split loose clusters (Clusters with relatively high SSE)
    3. Merge close clusters (Clusters with relatively low SSE)

Limitations of K-means

K-means has problems when clusters are of differing sizes, densities.

when the data contains outliers (not belonging to any cluster).

The similarity function is suitable or not.

4. Hierarchical Clustering

Do not have assume any particular number of clusters

Any desired number of clusters can be obtained by cutting the dendrogram at the proper level

Two main types of hierarchical clustering

1. Agglomerative clustering algorithm

Agglomerative: Start with the points as individual clusters and at each step, merge the closest part of clusters until only one cluster (or k clusters) left. (More popular)

Algorithm:

Compute the proximity/distance matrix

Let each point be a cluster

repeat

​ Merge the two closet clusters

​ update the proximity/distance matrix

until only a single cluster remains

2. Divisive clustering algorithm

Divisive: Start with one, all-inclusive cluster and at each step, split a cluster until each cluster contains a point (or there are k clusters)

Traditional hierarchical algorithms use a similarity or distance matrix to merge or split one cluster at a time

5. Inter-cluster Similarity

MIN: most similar (closest) points in the different clusters

MAX: two least similar (most distant) points in the different clusters

Group Average: Proximity of two clusters is the average of pairwise proximity between points in the two clusters.

Distance Between Centroids

6. Problems and Limitations

No objective function is directly minimized

Different schemes have problems with one or more of the following:

Sensitivity to noise and outliers Difficulty handling different sized clusters Breaking large clusters

Supervised Feature Extraction

1. Linear Discriminant Analysis

Linear discriminant analysis (LDA) finds most discriminative projection by maximizing between-class distance and minimizing within-class distance.

Class-specific mean vector \[ \mu_i = \frac{1}{n_i}\sum_{x\in C_i} x \] Class-specific covariance matrix \[ S_i=\frac{1}{n_i}\sum_{x\in C_i} (x-\mu_i)(x-\mu_i)^T \]

Total mean vector \[ \mu = \frac{1}{N}\sum_x x \] Within-class scatter \[ S_W=\sum_{i=1}^C \frac{n_i}{N}S_i=\sum_{i=1}^C P_iS_i \] where \(P_i\) weight of a specific class.

Between-class scatter \[ S_B=\sum_{i=1}^CP_i (\mu_i-\mu)(\mu_i-\mu)^T \] That means the sum of the distance of all class specific mean vector to the total mean vector.

Total covariance \[ S_T=\frac{1}{N}\sum_x (x-\mu)(x-\mu)^T=S_W+S_B \] Proof skipped.

Two classes

\[ y=w^Tx \]

Then we can get the scalar y, of all possible \(w\), we hope to choose the best that maximizes the separability. \[ \mu_i=\frac{1}{n_i}\sum_{x\in C_i}x\quad \tilde \mu_i=w^T\mu_i \] The distance between the two classes \[ J(w)=|\tilde \mu_1-\tilde \mu_2|=|w^T(\mu_1-\mu_2)| \] However, the distance can be infinitely large if no constraint to the chosen \(w^T\), therefore, we normalize the distance \[ J(w)=\frac{|\tilde \mu_1-\tilde \mu_2|^2}{\tilde S_1^2+\tilde S_2^2} \] where \(\tilde S_i^2\) is the Class-specific covariance matrix of \(\tilde y_i\), \(\tilde S_1^2+\tilde S_2^2\) is the Within-class scatter of \(\tilde y_i\)

Then we find \(\tilde S_1^2+\tilde S_2^2=w^TS_W w\) and \(|\tilde \mu_1-\tilde \mu_2|^2=w^TS_Bw\)

Then \[ J(w)=\frac{w^TS_B w}{w^TS_Ww} \] We take the derivative and let the derivative be zero (\((w^TS_Ww)^2\) is omitted), \[ (w^TS_W w) S_Bw-(w^TS_B w) S_Ww=0\\ S_Bw-J(w)S_Ww=0\\ S_W^{-1}S_Bw=J(w)w \] That means \(w^*\) is the eigenvector of \(S_w^{-1}S_B\)

For Multi-class \[ J(W)=\frac{|W^TS_BW|}{|W^tS_WW|}\\ (S_B-\lambda_i S_W)w_i^*=0 \] Then \(w_i^*\) are the largest eigenvectors corresponding to the largest eigenvalues.
