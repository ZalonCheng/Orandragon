3.1 Fundamental of Lyapunov Theory

Posted on 2018-10-02 Edited on 2019-08-26 In Applied Nonlinear Control

1. Nonlinear System and Equilibrium Points

  1. There are two methods to solve the stability of a system proposed by Lyapunov. Linearization method and direct method.

  2. Nonlinear system can be represented by \[ \dot{\mathbf x} = \mathbf f(\mathbf x,t) \] and if there is input \(u\) \[ \dot{\mathbf x}= \mathbf f(\mathbf x,\mathbf u,t) \\ u=g(\mathbf x,t) \] and if \(u\) depends on \(x\) and \(t\) ,we have \[ \dot{\mathbf x}= \mathbf f(\mathbf x,\mathbf g(\mathbf x,t),t) \] if there is no control signal, the first differential equation is used.

2. Autonomous and Non-Autonomous Systems

The system is said to be autonomous if \(\mathbf f(x)\) does not depend explicitly on time \(t\) \[ \dot{\mathbf x} = \mathbf{f}(\mathbf x) \] Otherwise, the system is called non-autonomous.

Obviously, LTI systems are autonomous and LTV are non-autonomous system

Autonomous systems and non-autonomous systems, time-varying systems and time invariant systems are the definition on the closed-loop dynamics. Therefore, they can be due to the plant of the control law.

The fundamental difference between autonomous system and non-autonomous systems is the former is independent of the initial time.

3. Equilibrium Points

==Equilibrium points== are the state of the system if once \(x(t)\) is equal to \(x^*\) , it remains to \(x^*\) for all future time.

Because of \[ \tag{3.1} \dot{\mathbf x} = \mathbf f(\mathbf x) \] We have \[ \mathbf f(\mathbf x^*)= \mathbf 0 \] For a LTI system \[ \dot{\mathbf x}=A\mathbf x\\ A\mathbf x^*=0 \] If \(A\) is nonsingular, that means there is no nontrivial solution for the equation, there is only trivial solution \(\mathbf x^*=0\) , there is only one equilibrium for the system and if \(A\) is singular, there are equilibriums same as the solutions for the above equation.​

And if the equilibrium is not at the origin, we can change transfer it to the origin. \[ \begin{array}{lll} \mathbf y = \mathbf x - \mathbf x^*\\ \mathbf x= \mathbf y + \mathbf x^* \\ {(\mathbf y + \mathbf x^*）'}=\mathbf f(\mathbf y + \mathbf x^*) \\ \dot{\mathbf y}=\mathbf f(\mathbf y + \mathbf x^*) \end{array} \]

==Nominal Motion==

In some practical problems, we are not concerned with stability around the equilibrium point but rather with the stability of the motion such as whether a system will remain close to its original motion trajectory if slightly perturbed away from it. We can show that this kind of motion stability problem around an equilibrium point, although the equivalent system is now non-autonomous.

$x^(t) $ is the solution of (3.1) corresponding to the initial value $x^(0)=x_0 $, and if there is a small disturbance let the $x(0)=x_0+x_0 $ and the solution of (3.1) is \(x(t)\) . \[ \begin{array}{} e = x(t)-x^*(t)\\ \dot{x^*}=f(x^*)& x(0)=x_0\\ \dot{x}=f(x) & x(0)=x_0+\delta x_0 \\ \dot{e}=f(x^*+e)-f(x^*) =g(e,t)& e(0)=\delta x_0 \end{array} \] We note that \(g(0,t)=0\) , therefore \[ \dot{e}=g(e,t) \] is a new system with state \(e\) and function \(g\)

Because of \(g(0,t)=0\) ,the origin is a equilibrium point.

Because of the \(x^*(t)\) on the right hand side, the system is non-autonomous and the equations are \[ \dot{e}=g(e,t) \] Instead of \[ \dot{e}=g(e) \] This will be discussed in the chapter 4 (according to the book), because it is difficult to analyze a non-autonomous system.

Example \[ m\ddot x+k_1x+k_2x^3=0\\ x_1=x\\ x_2=\dot x\\ \] Then we have \[ \dot x_1=x_2\\ m\dot x_2+k_1x_1+k_2x_1^3=0 \] Rearrange the equations we have \[ \dot x_1=x_2\\ \dot x_2=-\frac{k_1}{m}x_1-\frac{k_2}{m}x_1^3 \] Let us study the stability of the motion \(x^*(t)\) which starts from initial position \(x_0\) .

Then \[ \dot x_1^*=x_2^*\\ \dot x_2^*=-\frac{k_1}{m}x_1^*-\frac{k_2}{m}(x_1^*)^3\\ \] The above equations with initial position \(x_0\) \[ \dot x_1=x_2\\ \dot x_2=-\frac{k_1}{m}x_1-\frac{k_2}{m}x_1^3 \] The above equations with initial position \(x_0+\delta x_0\) \[ \dot e_1=e_2\\ \dot e_2=-\frac{k_1}{m}e_1-\frac{k_2}{m}(e_1+x_1^*(t))^3+\frac{k_2}{m}(x_1^*)^3 \] If systems are LTI systems, the nominal motion can be shown as \[ \dot{e}=Ae \]

4. Concepts of Stability

Let \(B_R\) denote the spherical region defined by \(||x||<R\)

Let \(S_R\) denote the sphere itself defined by \(||x||=R\)

==Stability== \[ \forall R>0, \exist r>0,||x(0)||<r\implies \forall t \ge0, ||x(t)||<R \] The system is stable.

Negation \[ \forall R>0, \exist r>0,||x(0)||<r\implies \exist t \ge0, ||x(t)|| \ge R \] The system is unstable.

==Asymptotic Stability==

The system must be stable and \[ \forall R>0,\exist r>0, ||x(0)||<r \implies t \rightarrow \infty ,x(t)\rightarrow 0 \] State converge does not imply stability, the above only prove the converge, but may not be stable.

==Exponential Stability== \[ \forall R>0,\exist r>0, ||x(0)||<r \implies\forall t>0, \exist \alpha\space\lambda,||x(t)||\le\alpha||x(0)||e^{-\lambda t} \] Exponential stability implies asymptotic stability

==Global Stability==

If the asymptotically (exponentially) stable holds for any initial states, it is called globally asymptotically (exponentially) stable.

5. Linearization and Local Stability

\[ \begin{array}{} \dot{x}&=&(\frac{\partial f}{\partial x})_{x=0}x+O(f(x)) \\ A &=& (\frac{\partial f}{\partial x})_{x=0} \\ \tag{3.2} \dot{x}&=&Ax \end{array} \]

That is called the linearization (linear approximation) of the original nonlinear system.

and for \[ \begin{array}{} \dot{x}=f(x,u) \\ \dot{x}=(\frac{\partial f}{\partial x})_{x=0,u=0}x+(\frac{\partial f}{\partial u})_{x=0,u=0}u+O(f(x,u))\\ A = (\frac{\partial f}{\partial x})_{x=0,u=0} \\ B = (\frac{\partial f}{\partial u})_{x=0,u=0} \end{array} \] If the control law of the system is \(u=u(x)\), which is autonomous, and \(u(0)=0\) \[ \begin{array}{} u=u(x)=\frac{\partial u}{\partial x}\vert_{x=0}u=Gx \\ G = \frac{\partial u}{\partial x}_{x=0}u\\ \dot{x}=f(x,u)=(A+BG)x \end{array} \] ==Lyapunov's Linearization Method==

If the linearized system is strictly stable, then the equilibrium point is asymptotic stable

If the linearized system is strictly unstable, then the equilibrium point is unstable

If the linearized system is marginally stable, one cannot make any conclusion. The nonlinear system can be stable and also can be unstable.

Example

Consider the system \[ \dot x_1=x_2^2+x_1\cos x_2\\ \dot x_2=x_2+(x_1+1)x_1+x_1\sin x_2\\ \] We hope to check the equilibrium point \(x=0\) .

First, linearize the system \[ \begin{array}{rcl} \frac{df_1}{x_1}|_{x=0} & = & (\cos x_2)|_{x=0}=1\\ \frac{df_1}{x_2}|_{x=0} & = & (2x_2-x_1\sin x_2)|_{x=0}=0\\ \frac{df_2}{x_1}|_{x=0} & = & (2x_1+1+\sin x_2)|_{x=0}=1\\ \frac{df_2}{x_2}|_{x=0} & = & (1-x_1\cos x_2)|_{x=0}=1\\ \end{array} \] Then we have \[ A= \begin{bmatrix} 1&0\\ 1&1\\ \end{bmatrix}\quad \] Example

Consider the following system \[ \dot x=ax+bx^5 \] After linearization \[ \dot x=ax \] Then we can say

  1. a<0, asymptotically stable
  2. a>0, unstable
  3. a=0, none conclusion

If the third case, we need to use direct method.

6. Lyapunov's Direct Method

If the total energy of the system is continuously dissipated, then the system will eventually settle down to an equilibrium point.

Example

Consider the mass-damper-spring system \[ m\ddot x+b\dot x|\dot x|+k_0x+k_1x^3=0 \] Then the energy of the system is the summarization of kinetic energy and potential energy. \[ V=\overbrace{\frac{1}{2}m\dot x^2}^{\text{Kinetic Energy}}+\overbrace{\int_0^{x}(k_0x+k_1x^3)dx}^{\text{Potential Energy}} \] Then we derivative the energy function with respect to t \[ \dot V=m\dot x\ddot x+(k_0x+k_1x^3)\dot x=-b|\dot x|^3\le0 \] Some conclusions

Zero energy corresponds to the equilibrium point

Asymptotic stability implies the convergence of mechanical energy to zero

Instability is related to the growth of energy

Positive definite functions and Lyapunov Functions

The energy functions have two properties.

1> It is strictly positive unless all state variables are zero. (Positive definite functions)

2> The function is monotonically decreasing when the states varying according to the state functions. (Lyapunov Functions)

Locally Positive Definite, if \(V(0)=0\) and in a ball \(B_{R_{0}}\) \[ x \neq 0 \implies V(x)>0 \] Globally Positive Definite, if the above property holds for the whole state space.

\(V(x)\) is negative definite if \(-V(x)\) is positive definite

\(V(x)\) is positive semi-definite if $V(x) 0 $ for \(x \neq 0\)

\(V(x)\) is negative semi-definite if \(-V(x)\) is positive semi-definite.

\[ \dot{V}=\frac{dV(x)}{dt}=\frac{\partial V}{\partial x}\frac{dx}{dt}=\frac{\partial V}{\partial x}f(x) \] If in a ball \(B_{R_{0}}\) , the function \(V(x)\) is positive definite and has continuous partial derivatives, and if derivative along any state trajectory of system is negative semi-definite \[ \dot{V}(x) \le 0 \] Then \(V(x)\) is said to be ==Lyapunov Function== for the system.

==Equilibrium Point Theorems==

Lyapunov Theorem for local stability

if, in a ball \(B_{R_{0}}\) ,there exists a scalar function V(x) with continuous first partial derivatives such that

\(V(x)\) is positive definite

\(\dot{V}(x)\) is negative semi-definite

then the equilibrium is stable.

If \(\dot{V}(x)\) is negative definite, then the equilibrium is asymptotically stable.

Proof is in another note.

Example: Local Stability

A simple pendulum with viscous damping is described by \[ \ddot \theta+\dot \theta+\sin \theta=0 \] Then we have state space equations \[ \dot x_1=x_2\\ \dot x_2=-x_2-\sin x_1 \] We choose a Lyapunov Function \[ V(x)=(1-\cos x_1)+\frac{x_2^2}{2} \] It can be easily verified that it is locally positive definite. \[ \dot V(x)= \begin{bmatrix} \frac{\partial V}{\partial x_1} & \frac{\partial V}{\partial x_2} \end{bmatrix} \begin{bmatrix} \frac{\partial x_1}{\partial t} \\ \frac{\partial x_2}{\partial t} \end{bmatrix} =\begin{bmatrix} \sin x_1 & x_2 \end{bmatrix} \begin{bmatrix} x_2 \\ -x_2-\sin x_1 \end{bmatrix}=-x_2^2<0 \] Actually, for this question, \(\dot V(x)\) consists two Jacobi Matrices.

Example: Asymptotic Stability

Consider the system \[ \dot x_1=x_1(x_1^2+x_2^2-2)-4x_1x_2^2\\ \dot x_2=4x_1^2x_2+x_2(x_1^2+x_2^2-2)\\ \]

\[ V(x)=x_1^2+x_2^2 \]

Then we have \[ \dot V(x)= \begin{bmatrix} 2x_1 & 2x_2 \end{bmatrix} \begin{bmatrix} \dot x_1\\ \dot x_2\\ \end{bmatrix}= 2(x_1^2+x_2^2)(x_1^2+x_2^2-2) \] \(\dot V(x)\) is locally negative definite, therefore the system is locally asymptotically stable.

Lyapunov Theorem for global stability

\(V(x)\) is positive definite

\(\dot{V}(x)\) is negative definite

\(V(x) \rightarrow \infty\) as \(||x|| \rightarrow \infty\)

Then the system is globally asymptotically stable. The last one condition is to ensure the closed contour of \(V(x)\)

Example: Globally Asymptotic Stability

Consider the system \[ \dot x_1=x_2-x_1(x_1^2+x_2^2)\\ \dot x_2=-x_1-x_2(x_1^2+x_2^2)\\ \] We choose the positive definite function \[ V(x)=x_1^2+x_2^2 \] Then we have \[ \dot V(x)=2x_1\dot x_1+2x_2\dot x_2^2=-2(x_1^2+x_2^2)^2 \] V(x) goes to infinity when x goes to infinity.

Therefore, the system is globally asymptotically stable.

Many Lyapunov Functions exist for the same system. e.g.

If V is an energy function for this system, then \[ V_1=\rho V^\alpha \] \(\rho\) is strictly positive definite and \(\alpha\) is a scalar. Then all of these energy functions are for this system.

==Invariant set theorems==

Asymptotic stability of a control system is usually very important property to be determined. However, the equilibrium point theorem just described are often difficult to apply to assert this property. The reason is that it often happens that \(\dot V\) is only semi-definite. That is the reason of existence of invariant set theorem. It can be used to determine the asymptotic stability of a control system with semi-definite \(\dot V\).

A set is an invariant set for a dynamic system if every system trajectory which starts from a point in G remains in G for all future time.

Local invariant set theorem

The invariant set theorem reflect the intuition that the decrease of a Lyapunov function V has to gradually vanish.

Assume

For some \(l>0\) , the region \(\Omega _l\) defined by \(V(x)<l\) is bounded

\(\dot{V}(x)\le 0\) for all \(x\) in \(\Omega_l\)

Let R be the set of all points in \(\Omega_l\) where \(\dot{V}(x)= 0\) , and \(M\) be the largest invariant set in \(R\) . Then every solution \(x(t)\) originating in \(\Omega_l\) tends to \(M\) as \(t \rightarrow \infty\) .

(if $(x)=0 $ for all future \(t\), then \(M=R\) )

That means, if M only contains the origin, then states will be converged to the origin. Therefore, the system is locally asymptotic stable. It is summarized in the following

\(V(x)\) is locally positive definite

\(\dot{V}\) is negative semi-definite

The set \(R\) defined by \(\dot{V}(x)=0\) only contains the origin.

Then the equilibrium is asymptotically stable.

Furthermore, the largest connected region of the \(\Omega_l\) (defined by \(V(x)<l\)) is a domain of attraction.

Globally invariant set theorem

\(\dot{V}(x) \le 0\) over the whole state space

\(V(x) \rightarrow \infty\) as \(||x|| \rightarrow \infty\)

Let R be the set of all points where \(\dot{V}(x) = 0\) , and M be the largest invariant set in R. Then all solutions globally asymptotically converge to M as \(t\rightarrow \infty\) .

Proof omitted needs knowledge of topology.

Example

mass-damper-spring system introduced in the beginning is marginally stable if we use Lyapunov Theory. We have \[ \dot V=m\dot x\ddot x+(k_0x+k_1x^3)\dot x=-b|\dot x|^3\le0 \] Then \(\dot x=0\) and \(x\) is free

Then we will use Invariant Set Theorem to prove that it is asymptotically stable.

We know that states will goes to largest invariant set M. We only need to show that only origin is in this set.

If there is another none zero point in this set, then \(x\neq0\) and \(\dot x=0\). If we check the acceleration, it is zero only when \(x=0\), which means none zero point is not in invariant set. Therefore, the system is asymptotically stable.

6. Existence of Limit Circle

N is the number of nodes, centers and foci enclosed by a limit circle.

S is the number of saddle points enclosed by a limit circle.

  1. Poincare

    If a limit circle exists in the second-order autonomous system, then \(N=S+1\)

    the proof is mathematically involved and the inference is that from this theorem, a limit circle must enclose at least one equilibrium point.

  2. Poincare-Bendixson

    If a trajectory of the second order autonomous system remains in a finite region \(\Omega\) , then one of the following is true

    1. the trajectory goes to an equilibrium point
    2. the trajectory tends to an asymptotically stable limit circle
    3. the trajectory is itself a limit circle.

  3. Bendixson

    For the nonlinear system, no limit circle can exist in a region \(\Omega\) of the phase plane in which \(\frac{\part f_1}{\part x_1}+\frac{\part f_2}{\part x_2}\) does not vanish and does not change sign.
